## 起点主站

### 读者画像

- **年龄**：18-35岁为主，男性占绝对多数
- **阅读习惯**：老书虫多，对质量要求高，愿意追长篇
- **付费意愿**：强，习惯订阅制
- **核心需求**：爽感+逻辑+世界观

### 节奏要求

- 开篇3章内必须建立核心冲突和金手指
- 前10章完成第一个小高潮
- 每30-50章一个大高潮
- 允许适度慢热，但不能断节奏
- 全书200万字起步，500万字不嫌多

### 题材偏好

- **男频主流**：都市/玄幻/仙侠/科幻/历史
- **女频偏弱**：起点女频存在感低，不如去晋江
- **热门子类**：系统流/重生/脑洞/无敌流/凡人流
- **冷门但有空间**：硬科幻/历史架空/悬疑

### 开篇策略

- 第一章必须出现金手指或核心设定
- 前3章建立世界观框架+主角目标
- 前10章完成第一个爽点循环
- 不要在开头大段设定介绍

### 更新策略

- 日更4000-6000字为基准
- 新书期爆更效果显著（日更万字以上）
- 稳定更新比间歇爆更更重要
- 关键剧情节点可以加更拉数据

## 番茄小说

### 读者画像

- **年龄**：25-45岁为主，下沉市场用户多
- **阅读习惯**：碎片化阅读，通勤/睡前为主
- **付费意愿**：低，以免费阅读+广告为主
- **核心需求**：轻松/解压/爽

### 节奏要求

- 开篇必须极快，第一章就要有爽点
- 每章结尾必须有钩子
- 章节短小（1500-2500字/章）
- 爽点密度极高，几乎每3章一个小爽
- 不允许慢热，读者耐心极低

### 题材偏好

- **男频热门**：都市赘婿/战神/神医/兵王
- **女频热门**：甜宠/重生/种田/宫斗/虐恋
- **通用热门**：重生/系统/穿越/逆袭
- **核心逻辑**：简单直白、爽感明确、不费脑子

### 开篇策略

- 第一段就要抓住注意力
- 第一章必须完成：人设+困境+金手指+第一个爽点
- 不要任何铺垫，直接进入核心剧情
- 标题党有效——书名和简介就是最大的钩子

## 刺猬猫

### 读者画像

- **年龄**：16-28岁，二次元文化受众
- **阅读习惯**：偏好日式轻小说风格，喜欢吐槽和玩梗
- **付费意愿**：中等，愿意为喜欢的作品付费
- **核心需求**：有趣/有梗/有脑洞

### 节奏要求

- 轻小说节奏，章节短（2000-3000字/章）
- 吐槽和玩梗是刚需
- 允许日常戏，但要有笑点
- 战斗/剧情节奏可以快，但日常要有趣

### 题材偏好

- **核心**：同人/二次元/轻小说风格原创
- **热门**：Fate/漫威/原神等同人、校园/异能/恋爱日常
- **特色**：吐槽系/玩梗系/反套路
- **冷门**：严肃向/纯文学向

## 飞卢

### 读者画像

- **年龄**：18-30岁，追求极致爽感
- **阅读习惯**：快节奏、高爽度、不费脑
- **付费意愿**：强，但要求爽感对等
- **核心需求**：爽/快/直白

### 节奏要求

- 极致快节奏，开篇即高潮
- 每章必须有爽点，不能有"水"章
- 章节短（2000字左右）
- 全书不宜过长（100-200万字）
- 爽点密度是所有平台最高的

### 题材偏好

- **男频绝对主流**：都市/玄幻/同人/系统
- **热门子类**：无敌流/签到流/曝光流/聊天群
- **核心逻辑**：金手指要大、爽点要密、节奏要快
- **女频**：基本没有女频市场

### 开篇策略

- 第一句话就要有信息量
- 第一章完成：金手指+第一个爽点+人设
- 不需要任何铺垫和世界观介绍
- 标题即卖点，简介即大纲

## 晋江

### 读者画像

- **年龄**：16-35岁，女性占绝对多数
- **阅读习惯**：重情感、重人设、重文笔，愿意追长篇
- **付费意愿**：强，习惯按章付费
- **核心需求**：情感共鸣/人设魅力/CP好嗑

### 节奏要求

- 允许慢热，但人设和情感要在线
- 前5章必须建立CP关系的基本框架
- 感情线推进要细腻，不能跳跃
- 每10-15章一个情感小高潮
- 全书50-150万字为主流

### 题材偏好

- **女频核心**：耽美/言情双线并行
- **言情热门**：甜宠/虐恋/宫斗/宅斗/重生/穿书/仙侠
- **耽美热门**：校园/电竞/仙侠/星际/ABO
- **特色**：文笔要求高、人设要求精、情感要求深
- **冷门但有空间**：悬疑/科幻/历史

### 女频核心

- 晋江是女频第一大站，没有之一
- 读者对文笔和人设的要求是所有平台最高的
- CP好嗑是第一生产力
- 标签系统非常重要——选对标签决定曝光
- 收藏和营养液是核心数据指标

### 开篇策略

- 人设和情感氛围优先
- 前3章必须让读者对CP产生期待
- 文笔要在线，不能太白话也不能太矫情
- 可以有铺垫，但必须有情感钩子

## 七猫

### 读者画像

- **年龄**：25-50岁，下沉市场为主
- **阅读习惯**：碎片化、追求轻松解压
- **付费意愿**：低，免费+广告模式
- **核心需求**：轻松/解压/代入感强

### 节奏要求

- 开篇要快，但不需要飞卢那么极端
- 每5章一个小爽点
- 章节适中（2000-3000字/章）
- 日常可以多，但不能无聊
- 女频节奏比男频稍慢，但也不能拖

### 题材偏好

- **男频热门**：赘婿/战神/神医/都市
- **女频热门**：甜宠/种田/重生/婆媳/逆袭
- **核心逻辑**：接地气、代入感强、不费脑子
- **特色**：婆媳/家长里短/生活流在七猫有独特市场

## 平台选择决策树

### 第一步：确定目标读者性别

```
你的目标读者是？
├── 男性 → 第二步A
└── 女性 → 第二步B
```

### 第二步A：男频平台选择

```
你的题材风格是？
├── 极致爽感/快节奏 → 飞卢
├── 二次元/轻小说/同人 → 刺猬猫
├── 下沉市场/简单直白 → 番茄/七猫
├── 质量向/长篇/世界观 → 起点主站
└── 不确定 → 起点主站（最稳的选择）
```

### 第二步B：女频平台选择

```
你的题材风格是？
├── 文笔向/人设精/CP向 → 晋江
├── 下沉市场/轻松解压 → 番茄/七猫
├── 甜宠/种田/生活流 → 七猫/番茄
├── 虐恋/宫斗/宅斗 → 晋江
└── 不确定 → 看下一条
```

### 第三步：细化选择

```
你的核心优势是？
├── 文笔好 → 晋江
├── 脑洞大 → 起点/刺猬猫
├── 爽点密 → 飞卢/番茄
├── 情感细腻 → 晋江
├── 接地气 → 七猫/番茄
└── 速度快/产量高 → 番茄/飞卢
```

### 第四步：交叉验证

选定平台后，检查以下条件：

1. **题材匹配** — 你的题材在该平台是否有人看？
2. **节奏匹配** — 你的写作节奏是否符合平台要求？
3. **竞争分析** — 该平台同类作品多不多？你有没有差异化？
4. **变现模式** — 该平台的变现方式你能否接受？
5. **长期发展** — 该平台是否适合你的长期规划？

| 晋江 | 女频第一站 | 中等 | 高 | 按章付费 | 极高 |

### 特殊情况处理

- **跨平台发布**：部分平台允许，注意合同条款
- **先免费后付费**：番茄/七猫的免费模式可以积累读者再转付费
- **多开**：精力允许可以不同平台开不同风格的书
- **新人建议**：先选一个平台深耕，不要同时多开
