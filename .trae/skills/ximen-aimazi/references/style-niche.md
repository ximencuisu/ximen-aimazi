# 小众题材风格模块

> 何时加载：用户选择轻小说/沙雕/赛博朋克/盘点文/模拟文/直播流等小众题材时。

## 轻小说/二次元风

### 核心定义

轻小说 = 以卖戏剧性人设为主的小说，尤其以卖各种美少女人设为核心。题材是否有日式风味反倒不是主要分类依据。

### 风格特征

- 人设即卖点：每个角色都有鲜明的"标签化"特质，读者一眼记住
- 日常感重于剧情推进：角色互动和日常碎片是主要内容
- 吐槽/内心戏密集：主角作为"吐槽役"，对荒诞事物的反应本身就是爽点
- 情绪轻快：即使有严肃剧情也用轻松笔调处理，不沉溺苦大仇深

### 写作要点

- 人设差异要极端化：傲娇/病娇/三无/天然呆，标签越鲜明越好
- 卖人设 > 卖剧情：剧情服务于展示角色魅力，而非相反
- 对话量 > 描写量：角色互动靠对话推进，减少大段心理独白
- 轻小说和传统网文的边界：轻小说允许节奏更散，但人设魅力必须持续输出

## 沙雕风

### 核心认知

沙雕风 = 用荒诞、反常规的叙事制造笑点，让读者"笑到停不下来"。在女频修真等题材中几乎成为标配。

### 风格特征

- 角色行为出人意料但逻辑自洽（不是乱来，是有脑子的荒诞）
- 严肃场景被角色的沙雕行为化解，反差制造笑点
- 宗门日常 > 战斗升级：沙雕宗门风中，升级只是不起眼的元素
- 群像感强：不是一个人搞笑，而是一群人互相衬托

### 写作要点

- 沙雕不等于降智：角色的行为要有自己的逻辑，只是这个逻辑很离谱
- 沙雕风的核心风险：纯沙雕没有情感深度，读者笑完就忘，需要有真情实感的底色
- 适合的题材：修真/玄幻/都市日常/轻小说
- 不适合的题材：严肃正剧/虐文/恐怖（氛围冲突）

## 赛博朋克风

### 风格特征

- 高科技低生活的质感：霓虹、义体、数据流、阶级固化
- 信息密度大：大量世界观细节需要自然融入叙事，不能写说明文
- 冷硬基调：疏离感、压抑感、技术异化感贯穿始终
- 边界感极强：稍有偏离就会"不对味"

### 写作要点

- 赛博朋克是"见光死"题材：大众认知有限，核心受众小众但忠诚
- 世界观细节 > 剧情复杂度：赛博朋克读者来看的是世界观氛围
- 必须在开头几章建立足够的氛围感：雨夜、霓虹、义体改造、虚拟网络
- 避免写披着赛博皮的玄幻：力量体系要服从科技逻辑，不是换个皮的修仙

## 盘点文/信息差文

### 核心机制

盘点文 = 通过主角掌握的独有信息（历史/文化/非遗/游戏等）向不知情的围观者展示，制造震惊和装逼效果。

### 风格特征

- 知识类信息差是核心爽点：读者和主角一起"碾压"不知情的世界
- 单元结构：一个盘点对象 = 一个装逼单元，循环推进
- 共鸣优先：盘点方向必须选择读者能产生民族/文化/专业共鸣的内容
- 难度低、上手快：找对切入方向就能写

### 写作要点

- 盘点方向的选择决定成败：历史/非遗/国潮/游戏/科技，要选有共鸣的
- 每个盘点单元需要：铺垫环境的不了解 → 主角展示 → 围观者震惊 → 信息差抹平
- 可以与其他题材缝合：盘点+游戏文、盘点+穿越文、盘点+娱乐圈
- 注意边界感：盘点文的节奏是信息差驱动，不能变成纯科普

## 模拟文

### 核心结构

模拟文 = 通过"模拟"获得奖励和信息差，再在现实中推进主线、满足期待。

### 单元故事结构

```
模拟制造情绪缺口 → 模拟中获得奖励（形成信息差）→ 现实满足期待
```

### 卷纲结构

1. 模拟得知大危机/大期待（支撑整卷的核心悬念）
2. 多次【模拟+得奖励+现实推进】的小单元循环
3. 最终达成/解决核心期待

### 关键要点

- 模拟中的信息差 = 现实装逼的筹码
- 模拟内容可以展示"如果不干预会怎样"的后果，强化紧迫感
- 奖励分层：小模拟给小奖励（升级资源），大模拟给关键信息（剧情转折）
- 模拟不是全知：保留未知变量，不能让主角通过模拟知道一切

## 直播流

### 核心机制

直播流 = 以直播场景为框架，融合搞钱、装逼、对抗、整活多种爽点。

### 爽点构成

| 爽点来源 | 具体表现 |
|----------|----------|
| 搞钱 | 直接收打赏/礼物，金钱数字直观增长 |
| 对抗 | PK、打榜、与同行竞争 |
| 身份马甲 | 观众中有隐藏大佬，弹幕信息差制造惊喜 |
| 整活 | 抽象/搞笑的直播内容吸引围观 |
| 才艺展示 | 主角独特能力的展示窗口 |

### 写作要点

- 目标感明确：每场直播有具体目标（圈够X钱/涨X粉），给读者确定预期
- 直播间弹幕 = 即时反馈系统：相当于内置的装逼观众，不需要额外设计
- 可以融合世情：每场直播接待一个客人/处理一件事 = 一个单元世情故事
- 注意节奏：不能一直直播，需要现实线穿插推进
