# 镜头式写作模块

> 何时加载：需要提升画面感、设计场景切换、或用户选择镜头式写作风格时。

## 镜头式写作法

忘掉全知视角，你是跟在主角身后的摄像机。读者只能看到主角看到的东西。

### 核心原则

1. **只写能看到/听到/闻到/摸到的**：不写「他心里想」「他感到」
2. **对话即剧情**：每句对话都要推进剧情、塑造人物、或传递信息
3. **动作代替描述**：「他很紧张」→「他的手在抖」
4. **手机屏幕优先**：短句、频繁换行、每段不超过 3 句

### 禁用清单

**禁用视角**
- ❌ 全知叙述：「此时的他还不知道，三天后…」
- ❌ 上帝视角：「如果她知道真相…」
- ❌ 心理分析：「他内心深处其实…」

**禁用词汇**
仿佛、好像、犹如、一丝、一抹、深吸一口气、缓缓、不禁、眼中闪过、嘴角勾起、眉头微皱、不容置疑、不易察觉

**禁用句式**
- 「...，带着...」（如：他说，带着一丝无奈）
- 「XX说道」→ 用动作神态替代
- 「他/她知道...」→ 直接展示，不告诉
- 「像XX一样的XX」→ 陈词滥调的比喻

### 替代技巧

| 禁用 | 替代 |
|------|------|
| 他感到紧张 | 他的手在抖 |
| 她很伤心 | 她转过身，肩膀微微颤动 |
| "好的。"他说道 | "好的。"他点了根烟 |
| 他深吸一口气 | 他把烟摁灭在烟灰缸里 |
| 她眼中闪过一丝惊讶 | 她的手停在了半空 |
| 他不禁笑了 | 他的嘴角翘了一下 |

### 对话写作规范

**每句对话必须满足至少一个：**
1. 推进剧情（透露新信息、做出决定、发生冲突）
2. 塑造人物（展现性格、背景、价值观）
3. 制造张力（威胁、隐瞒、试探）

**对话格式**
- 用动作打断长对话
- 每次说话不超过 3 句
- 用神态/动作代替「他说」「她说」

**对话时机**
- 长篇幅对话要塞在期待点和爽点之间——读者因为期待爽点会容忍中间的对话
- 对话本身可以强化驱动力、制造信息交流受阻、脱离读者预期
- 无意义的聊天和谈情说爱可以省略脑补
- 把部分信息转化为情节、心理描写、旁白、环境、动作等表达
- 适度将"一问一答"模式转化为"一个人主动说，另一个人给反应"

**对话情绪五级递增**

写矛盾冲突时，对话越不礼貌，情绪越强烈：

1. **客观陈述事实** — 最弱
2. **客观陈述 + 提出建议** — 礼貌但有力
3. **主观指责** — 开始有冲突感
4. **主观指责 + 强制命令** — 强情绪
5. **主观指责 + PUA抬升自己** — 最强

最强话术：打着"为你好"的幌子，句句不离关心，但句句都是嫌弃、指责、厌恶

**对话拉扯技巧**
- 一问一答没有情绪波动；要像打斗场景一样让对话有"出招"感
- 对话要有人物性格、隐藏意图、情绪递进，而非直奔主题
- 中间任何一个片段都能结束对话，字数合适即可

**好的对话**
```
"钱呢？"

王少往椅背一靠，翘起二郎腿。"什么钱？"

"上个月的三十万。"林风往前一步。

王少的手指敲了敲桌面。"我说了吗？我不记得了。"
```

**差的对话**
```
"你还欠我三十万块钱，是上个月借的，答应这个月还的。"林风说道。

"哦，那个钱啊，我最近手头有点紧，能不能再等等？"王少不好意思地说道。

"不行，我已经等了一个月了。"林风坚定地说道。
```

### 三机位法

场景写作的三机位交替法，用于控制叙事节奏和画面感：

| 机位 | 视角 | 内容 | 功能 |
|------|------|------|------|
| 机位1 | 近景 | 主角的动作、表情、心理 | 推进主线，读者代入 |
| 机位2 | 远景 | 配角反应、环境变化、围观者 | 扩展信息面，制造反差 |
| 机位3 | 旁白 | 设定补充、背景交代 | 适时补充必要信息 |

**机位交替原则**
- 核心1和2，机位3穿插使用
- 设定都由冲突引出，无关设定不要乱写
- 校对时检查：机位1多了排2，机位2多了排1，穿插3做画外音补充
- 可以主动增删镜头来控制节奏

### 场景与分镜基础

**场景划分原则**
- 场景划分是分镜的基础——不懂场景直接上分镜会显得凌乱
- 一个场景 = 一个地点 + 一个时间段 + 一组人物
- 场景内必须发生至少一个有意义的事件（推进剧情/深化关系/揭示信息）
- 场景结束要有钩子（新信息/新悬念/新冲突）

**视角转换控制**
- 视角转换不要过于频繁——描写一个完整情节后再进入下一个情节点
- 衔接过渡要自然，用相似物/五感/情绪做转场
- 单场景内尽量保持同一视角

**画面感要素**
- 三种画面层次交替：环境→人物动作→特写细节
