# 文风圣经

用于长篇网络小说的项目级文风约束。
默认原则：整本书固定 1 个基础文风，只允许卷级微调，不在每章重新选文风。

---

## 当前项目文风

_（默认推荐基线：适用于玄幻 / 修仙长篇网络小说，可在此基础上微调）_

- **基础文风**：A 热血爽文风
- **辅助文风**：C 古风仙侠风（只吸收意境、法宝/境界表述、天地异象）
- **混搭比例**：70 / 30
- **作家技法参考**：主参考 我吃西红柿 + 辅参考 辰东
- **当前生效范围**：整本

## 长篇默认规则

1. 整本书固定 1 个基础文风。
2. 最多叠加 1 个辅助文风，建议占比不超过 30%。
3. 作家技法参考最多 `1 位主参考 + 1 位辅参考`。
4. 只在开书、换卷、平台变化、明显风格漂移时调整。
5. 写正文前优先读取本文件，不要每章重选文风。

## 目标读感

- **目标平台**：起点 / 番茄 / 男频长篇连载平台
- **目标读者**：追更型玄幻 / 修仙男频读者，偏好升级、夺宝、秘境、宗门、飞升
- **阅读体感**：稳中带燃，仙气包裹爽感，世界越写越大但不虚
- **核心承诺**：境界成长 + 机缘夺取 + 势力升级 + 远期谜团

## 基础文风方案

- **主文风**：热血爽文风
- **辅助元素**：古风仙侠风的意境、法宝命名、天地异象、修行质感
- **人称/叙述距离**：第三人称近距离，默认贴主角，关键章允许短时切群像
- **段落节奏**：短中段落混合；战斗和破局用短段，修行/探索/秘境推进用中段，默认不超过 4 行
- **对话密度**：中；对话要有身份差、境界差、立场差，不写现代插科打诨式闲聊
- **环境描写密度**：中；优先写灵压、地势、异象、遗迹感，只保留能影响冲突和气氛的景物
- **说明/设定解释密度**：中低；规则跟着争夺、修行、试炼自然露出，不集中讲课
- **章末钩子强度**：高；优先保留“新秘境 / 新传承 / 新仇敌 / 新上界线索”

## 作家技法参考

- **主参考**：我吃西红柿
- **辅参考**：辰东
- **使用比例**：7 : 3

### 提炼的可迁移技法

- **节奏**：以“修行目标 / 资源争夺 / 秘境试炼 / 势力冲突”形成阶段闭环，卷与卷之间持续抬升
- **爽点兑现**：每次突破、夺宝、破局、斩敌、收传承都要立刻改变主角身份、战力或处境
- **世界观展开**：先宗门、王朝、界域，再古史、禁区、上界真相，层层抬升但每层都和当前利益绑定
- **对话气质**：简洁、克制、有位阶感，不写满屏大白话，也不堆砌假古文
- **情绪推进**：大战前先攒因果、师承、同门、旧仇、宿命感，再爆高潮
- **章末钩子**：优先用“下一份机缘 / 下一层真相 / 下一位更强对手 / 下一道未还因果”收章

### 禁止误用

- 不逐句模仿
- 不照搬代表桥段
- 不复制专属名词体系

## 全书硬约束

- **必须保持**：境界梯度清楚、资源来源清楚、功法法宝有来路、突破与付出成正比、每章有实质推进
- **禁止出现**：连续空转修炼、长段设定说明书、现代网络热梗乱入、境界名称漂移、反派只会叫嚣、机缘只靠天降
- **去AI味额外要求**：少写空泛“大道玄而又玄”，多写具体压力、异象、代价、身体反应、环境变化
- **平台适配要求**：开篇 20 章必须完成第一次机缘兑现和第一次反压；每 20-30 章至少出现一次秘境/势力/传承节点

## 卷级微调记录

| 卷/阶段 | 微调目标 | 允许变化 | 禁止变化 |
|---------|----------|----------|----------|
| 开篇期（1-30章） | 快立处境、快给机缘、快见第一次翻盘 | 钩子更硬、解释更少、敌我更清晰 | 禁止长时间闭门修炼，禁止开篇就写散世界线 |
| 成长期（30-120章） | 稳定升级、扩宗门/家族/界域、拉高世界感 | 势力线更厚、秘境更多、古史伏笔增加 | 禁止连续多章只讲设定，禁止突破只涨数值 |
| 飞跃卷 / 大高潮卷 | 冲突加压、集中回收、拉高宿命感 | 异象更强、句长更短、压迫感更高 | 禁止彻底换风格，禁止只靠喊招式名堆高潮 |

## 临时章节覆盖

> 仅用于特殊章节，例如大高潮、审讯章、恐怖章、感情爆发章。
> 用完即失效，避免把长篇主文风写散。

| 章节 | 覆盖目标 | 生效原因 | 用后是否回归主文风 |
|------|----------|----------|--------------------|
| 渡劫 / 突破章 | 提高异象、代价感、天地压迫感 | 大境界突破 / 渡劫 | 是 |
| 秘境探索章 | 提高遗迹感、未知感、禁忌感 | 新秘境 / 古遗迹 / 传承地 | 是 |
| 决战章 | 提高爆发感、速度感、宿命感 | 宗门大战 / 生死战 / 反杀 | 是 |
| 情感 / 师徒章 | 提高关系张力与克制感 | 传承、诀别、师徒、同门情义 | 是 |
