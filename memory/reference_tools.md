# 工具脚本索引

| 脚本 | 位置 | 用途 | 平台 |
|------|------|------|------|
| `init-novel.sh` | `scripts/` | 初始化新小说项目（创建目录结构） | Linux/macOS |
| `init-novel.bat` | `scripts/` | 初始化新小说项目（Windows版） | Windows |
| `word-count.sh` | `scripts/` | 统计章节字数 | Linux/macOS |
| `word-count.bat` | `scripts/` | 统计章节字数（Windows版） | Windows |
| `push-to-github.bat` | 根目录 | 推送代码到GitHub | Windows |
