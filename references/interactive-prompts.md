# 交互提示词模板

各阶段的标准交互提示词，用于引导用户完成12阶段创作流程。

---

## Phase 1：灵感生成

### 启动提示词

```
欢迎使用网文创作助手！我们开始 Phase 1：灵感生成。

请告诉我您想写什么类型的小说，哪怕只有一句话也行。
例如："写一个都市修仙的爽文"、"重生回高中逆袭成商业大亨"
```

### 快速确认提示词

```
根据您的方向"{{用户方向}}"，我为您生成以下创作方案：

**方案一**：《{{书名1}}》
- 一句话简介：{{一句话简介1}}
- 核心爽点：{{爽点1}}
- 目标读者：{{读者1}}

**方案二**：《{{书名2}}》
- 一句话简介：{{一句话简介2}}
- 核心爽点：{{爽点2}}
- 目标读者：{{读者2}}

**方案三**：《{{书名3}}》
- 一句话简介：{{一句话简介3}}

```

## Phase 4：创作参数配置（交互式）

### AskUserQuestion 格式

```
现在进入 Phase 4：创作参数配置。请确认以下参数：

**参数1：章节数量**
- A. 50章（短篇/试水）
- B. 100章（标准）
- C. 200章（长篇）
- D. 500章（超长篇）
- E. 自定义

**参数2：女角色数量**
- A. 1个（唯一女主）
- B. 2个（双女主）
- C. 3个（经典三女主）
- D. 5个（后宫向）
- E. 自定义

**参数3：擦边等级**
- A. 无（纯剧情）
- B. 少量（偶尔暧昧，每10-20章1次）
- C. 中量（定期安排，每4-6章1次）
- D. 大量（频繁出现，每2-3章1次）

**参数4：写作风格**
- A. 热血爽文风（短句密集、节奏快、打脸干脆）
- B. 都市现实风（口语化、生活感、细节真实）
- C. 古风仙侠风（半文半白、意境优先、节奏沉稳）
- D. 悬疑惊悚风（氛围压抑、悬念迭起、细节暗示）
- E. 轻松日常风（吐槽密度高、节奏舒缓、互动有爱）
- F. 自定义文风

请分别选择4个参数（如"B/C/B/A"），或逐项确认。
```

### 擦边风格追问（选择非"无"时）

```
您选择了"{{擦边等级}}"擦边等级，请选择描写风格：

- A. 含蓄（暗示为主，留白多）
- B. 直白（不回避，但不低俗）
- C. 文艺（意境渲染，情感驱动）
- D. 暴力美学（冲突感强，张力优先）
- E. 自定义

请选择擦边描写风格？
```
