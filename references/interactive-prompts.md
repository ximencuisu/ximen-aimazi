# 交互提示词模板

各步骤的标准交互提示词，用于引导用户完成创作流程。

---

## 第一步：灵感生成

### 启动提示词

```
欢迎使用小说创作助手！我们开始第一步：灵感生成。

请回答以下问题（或直接告诉我您想写什么类型的小说）：
```

### 快速确认提示词

```
根据您的方向"{{用户方向}}"，我为您生成以下创作方案：

**书名备选**：
1. 《{{书名1}}》
2. 《{{书名2}}》
3. 《{{书名3}}》

**一句话简介**：
{{一句话简介}}

**核心爽点**：
{{爽点描述}}

**目标读者**：
{{目标读者}}

---
请选择一个方案或进行调整？
```

### 深度收集提示词（用户方向模糊时）

```
为更好生成创意方案，请补充以下信息：

1. **题材**：[玄幻/都市/仙侠/其他]
2. **核心卖点**：[系统流/穿越重生/逆袭打脸/甜宠/悬疑]
3. **目标字数**：[30万/50万/100万+]
4. **更新频率**：[日更/周更/不定期]

您可以直接告诉我"跳过"，我会根据您的方向直接生成。
```

## 第四步：大纲生成

```

**基本信息**：
- 题材：{{题材}}
- 预计章节：{{章节数}}章
- 每章字数：约{{每章字数}}字

**三幕式结构**：
- 第一幕（第1-{{第一幕终}}章）：建立世界，确立目标
- 第二幕（第{{第二幕始}}-{{第二幕终}}章）：成长试炼，遭遇挫折
- 第三幕（第{{第三幕始}}-{{第三幕终}}章）：最终对决，达成目标

---
请确认大纲结构，我将生成详细章节规划。
```
