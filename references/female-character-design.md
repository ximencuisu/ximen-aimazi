## 女主类型

### 白莲花

- **核心特征**：外表柔弱内心坚韧，看似无害实则有底线
- **适合题材**：虐文/宫斗/宅斗
- **人设要点**：
  - 柔弱是表象，坚韧是内核——关键时刻绝不退缩
  - 善良但不圣母，有原则有底线
  - 成长弧线明显：从隐忍到反击
- **爽点来源**：以柔克刚/弱者逆袭/隐忍后的爆发
- **写作禁忌**：不能真傻真弱，否则读者会弃文；不能圣母到没底线

### 黑莲花

- **核心特征**：表面无害实则心机深，善于伪装和借势
- **适合题材**：宅斗/宫斗/重生
- **人设要点**：
  - 外表人畜无害，内心算无遗策
  - 善于利用别人的弱点，但有自己的底线
  - 不是纯粹的恶，而是被环境逼出的生存智慧
- **爽点来源**：扮猪吃虎/反杀/算计得逞/真面目曝光
- **写作禁忌**：不能黑得没底线，要有让人共情的理由；算计不能太刻意

### 女强型

- **核心特征**：实力碾压不靠男人，独立自主有主见
- **适合题材**：玄幻/都市/经商
- **人设要点**：
  - 实力是自己的，不是男人给的
  - 可以有感情线，但不能依附于男人
  - 强大但有软肋，不是无情机器
- **爽点来源**：实力碾压/独立解决危机/不需要拯救
- **写作禁忌**：不能强到没朋友、没弱点；感情线不能太弱也不能喧宾夺主

### 甜软型

- **核心特征**：可爱治愈系，让人想保护
- **适合题材**：甜宠/日常/校园
- **人设要点**：
  - 软萌但不蠢，可爱但有主见
  - 善于发现生活中的小确幸
  - 关键时刻有自己的坚持
- **爽点来源**：被宠/被偏爱/小机灵/反差萌
- **写作禁忌**：不能蠢到让人生气；不能作到让读者想打人

### 清冷型

- **核心特征**：高岭之花，不食人间烟火
- **适合题材**：仙侠/古言/玄幻
- **人设要点**：
  - 外表清冷内心有温度，反差是关键
  - 不轻易动情，一旦动情就是全部
  - 强大但不张扬，低调但有实力
- **爽点来源**：动情反差/为一人破例/冷面下的温柔
- **写作禁忌**：不能冷到没感情；动情不能太快太突然

## 男主类型

### 霸总

- **核心特征**：强势占有欲，说一不二
- **适合题材**：甜宠标配
- **人设要点**：
  - 对外冷酷霸道，对女主温柔独宠
  - 占有欲强但不能控制欲变态
  - 强大是基础，偏爱是灵魂
- **经典台词风格**："你是我的""谁准你看别人""我说不行就是不行"
- **写作禁忌**：不能霸道到不尊重女主；占有欲和控制欲要有边界

### 暖男

- **核心特征**：温柔体贴，润物无声
- **适合题材**：治愈系/校园/日常
- **人设要点**：
  - 温柔不是软弱，是有力量的包容
  - 默默守护，关键时刻挺身而出
  - 细节见真心——记住女主的每个小习惯
- **经典台词风格**："我一直在""没关系，有我""你开心就好"
- **写作禁忌**：不能暖到没主见；不能对所有人都一样暖（要偏心）

### 病娇

- **核心特征**：偏执占有，爱到扭曲
- **适合题材**：虐恋向/暗黑系
- **人设要点**：
  - 爱是执念，得不到就毁灭
  - 表面正常，内心疯狂
  - 对女主的占有欲超出常理
- **经典台词风格**："你只能是我的""想逃？试试看""我不会让你离开"
- **写作禁忌**：不能只有病没有爱；要有让人心疼的过去；行为要有底线

### 腹黑

- **核心特征**：表面温和实则算计，运筹帷幄
- **适合题材**：宅斗/宫斗/权谋
- **人设要点**：
  - 笑着把人算计了，对方还感恩戴德
  - 对女主的算计是保护不是伤害
  - 智商在线，算无遗策
- **经典台词风格**："我怎么会害你呢""放心，都安排好了""有趣"
- **写作禁忌**：不能算计女主（除非是为了保护）；不能聪明到没对手

### 高岭之花

- **核心特征**：清冷禁欲，不近女色
- **适合题材**：仙侠/古言
- **人设要点**：
  - 万年冰山只为一人融化
  - 禁欲是表象，深情是内核
  - 动情后的反差是最大看点
- **经典台词风格**："……""不必""你……与其他人不同"
- **写作禁忌**：不能冷到读者没耐心；动情要有铺垫不能突然

## CP模式

### 甜宠

- **模式特征**：全程高甜，冲突来自外部
- **感情线**：从甜到更甜，螺旋上升
- **冲突来源**：外部势力（情敌/家族/误会），不伤CP感情
- **节奏**：日常甜度60%+，小波折20%，解决+更甜20%
- **关键**：甜不能腻，要有变化；每章至少一个心动瞬间

### 虐恋

- **模式特征**：甜蜜→误会→虐心→和好
- **感情线**：先甜后虐再甜，U型曲线
- **冲突来源**：CP之间的误解/背叛/立场对立
- **节奏**：甜30%→虐40%→和好30%
- **关键**：虐的前提是甜够；和好要有过程不能一秒原谅

### 双向奔赴

- **模式特征**：双方都有成长，互相成就
- **感情线**：平行成长→交汇→携手
- **冲突来源**：各自成长路上的障碍
- **节奏**：各自线40%+交汇线20%+携手线40%
- **关键**：双方戏份要均衡；成长要同步不能一个飞一个停

### 追妻火葬场

- **模式特征**：男追女，先虐后甜
- **感情线**：男伤女→女离开→男追回
- **冲突来源**：男主的前期伤害+女主的信任危机
- **节奏**：伤害20%→离开10%→追回50%→和好20%
- **关键**：追的过程要够苦够诚；女主不能太容易原谅

### 先婚后爱

- **模式特征**：从陌生到相爱
- **感情线**：陌生→磨合→心动→相爱
- **冲突来源**：性格差异/生活习惯/外部阻力
- **节奏**：陌生30%→磨合30%→心动20%→相爱20%
- **关键**：婚后日常要写好；心动瞬间要有铺垫

## 配角设计

### 绿茶

- **功能**：制造冲突、衬托女主、推动剧情
- **设计要点**：表面温柔善良实则心机深，善于在男人面前装可怜
- **下场**：必须被揭穿/打脸，读者最期待的场景
- **注意**：不能太蠢，否则打脸没快感；要有自己的逻辑和动机

### 白月光

- **功能**：制造误会、考验CP、推动感情
- **设计要点**：男主心中的完美前任，是女主最大的不安来源
- **走向**：要么真完美让女主成长，要么假完美被揭穿
- **注意**：不能喧宾夺主；存在感要恰到好处

### 闺蜜

- **功能**：情感支撑、剧情助攻、调节氛围
- **设计要点**：真心为女主好，有自己的感情线
- **注意**：不能抢女主风头；不能变成工具人

### 婆婆

- **功能**：制造阻力、考验CP、反映社会现实
- **设计要点**：有自己的一套逻辑，不是纯粹的恶
- **走向**：要么被女主感化，要么被男主挡回去
- **注意**：不能脸谱化；要有让人理解的立场

### 情敌

- **功能**：催化感情、制造危机、衬托女主
- **设计要点**：有自己的魅力，不是纯粹的炮灰
- **注意**：高级情敌让读者也纠结，低级情敌只是打脸素材

## 女频角色关系网设计法

### 设计原则

1. **以女主为中心** — 所有人物关系围绕女主展开
2. **每条关系线都有功能** — 推动剧情/衬托人设/制造冲突
3. **关系网要有层次** — 核心圈（CP/闺蜜）→ 中间圈（家族/同事）→ 外围圈（路人/过场）
4. **关系要动态变化** — 敌人变盟友、朋友变情敌，保持新鲜感

### 设计步骤

**第一步：确定核心关系**
- 女主与男主的关系模式（见CP模式）
- 这是全书的主轴，其他关系都围绕它

**第二步：设计对立关系**
- 谁是女主的主要对手？（绿茶/婆婆/情敌）
- 对手的动机要合理，不能为反对而反对

**第三步：设计支撑关系**
- 谁是女主的盟友？（闺蜜/家人/贵人）
- 盟友要有自己的故事，不能只是工具人

**第四步：设计催化关系**
- 谁能推动CP感情发展？（白月光/情敌/助攻）
- 催化剂要适时出现，不能太早也不能太晚

**第五步：画出关系网**
- 用图表标明每对关系：友好/对立/暧昧/利用
- 检查是否有孤立角色（与主线无关的人要删掉）
- 检查关系密度（核心圈不超过5人）

### 关系网常见模式

```
        [白月光]——暧昧——[男主]——CP——[女主]——闺蜜——[闺蜜]
           |                |                |
         制造不安         追妻/宠妻        支撑/助攻
           |                |                |
        [情敌]          [婆婆/家族]      [绿茶]
```

### 关系网动态设计

- 每20-30章引入一个新关系或改变一个旧关系
- 关系变化要有触发事件，不能无缘无故
- 好的关系可以变坏，坏的关系可以变好
- 每次关系变化都要带来新的剧情可能
