# 章节质量评分系统

自动评估章节质量，提供可量化的分数和改进建议。

---

## 评分维度

| 维度 | 权重 | 分数范围 | 说明 |
|------|------|---------|------|
| **爽点设置** | 25% | 0-10 | 打脸/突破/逆转等爽感设计 |
| **情节连贯性** | 20% | 0-10 | 与前文设定、角色状态一致 |
| **节奏把控** | 15% | 0-10 | 铺垫→冲突→爆发→钩子 |
| **悬念钩子** | 15% | 0-10 | 章末悬念吸引力 |
| **对话质量** | 10% | 0-10 | 角色对话个性化程度 |
| **角色塑造** | 10% | 0-10 | 人物行为合理、有个性 |
| **字数达标** | 5% | 0-10 | 字数是否符合要求 |
| **去AI味** | 10% | 0-10 | 无AI生成痕迹，语言自然 |

**总分 = Σ(维度分数 × 权重)**

---

## 评分标准详解

### 1. 爽点设置（25%）

| 分数 | 标准 |
|------|------|
| 9-10 | 有明确的打脸/突破/逆转，爽感强烈，让人读完很爽 |
| 7-8 | 有爽点设计，但不够强烈或节奏略慢 |
| 5-6 | 有爽点但铺垫太长，爽感被稀释 |
| 3-4 | 爽点模糊，读完没什么感觉 |
| 0-2 | 完全没有爽点，平淡如水 |

**自检问题**：
- 本章主角做了什么让读者觉得"爽"的事？
- 被打脸的反派反应是否够夸张？
- 众人的震惊是否渲染到位？

### 2. 情节连贯性（20%）

| 分数 | 标准 |
|------|------|
| 9-10 | 完全符合前文设定，无任何矛盾 |
| 7-8 | 偶有轻微不一致，但不影响阅读 |
| 5-6 | 有明显矛盾但可通过解释圆回 |
| 3-4 | 存在设定冲突，需要修改 |
| 0-2 | 严重穿帮（已死角色出场、等级矛盾等） |

**自检问题**：
- [ ] 角色等级/状态与 CHARACTERS.md 一致
- [ ] 地点描写与 LOCATIONS.md 一致
- [ ] 未违背 STORY_BIBLE.md 中的设定
- [ ] 承接了上章结尾的剧情

### 3. 节奏把控（15%）

| 分数 | 标准 |
|------|------|
| 9-10 | 铺垫简洁→冲突快速升级→高潮爆发→干净收尾 |
| 7-8 | 节奏基本正确，但某环节稍拖沓 |
| 5-6 | 节奏尚可，但不够紧凑 |
| 3-4 | 节奏混乱，高潮来得太突然或太拖沓 |
| 0-2 | 完全没节奏，要么全程平要么全程嗨 |

**自检问题**：
- 开篇是否在 200 字内进入正题？
- 冲突是否逐步升级而非一步到位？
- 高潮是否有足够的铺垫才爆发？

### 4. 悬念钩子（15%）

| 分数 | 标准 |
|------|------|
| 9-10 | 钩子设计精妙，让人忍不住点下一章 |
| 7-8 | 有悬念但不够吸引人 |
| 5-6 | 有钩子但太常规，缺乏新意 |
| 3-4 | 钩子太弱或太明显 |
| 0-2 | 章末没有任何悬念 |

**自检问题**：
- 章末留下了什么悬念？
- 这个悬念是否足够让读者"追更"？
- 是"怎么了"还是"为什么"？后者更强

### 5. 对话质量（10%）

| 分数 | 标准 |
|------|------|
| 9-10 | 对话个性化强，每个角色说话风格不同 |
| 7-8 | 对话基本有区分，偶有同质化 |
| 5-6 | 对话能推进剧情，但角色区分不明显 |
| 3-4 | 对话生硬，像在念说明书 |
| 0-2 | 对话毫无个性，所有人一个腔调 |

**自检问题**：
- 主角说话风格是否鲜明？（比如：沉稳/傲娇/幽默）
- 反派说话是否够嚣张？配角是否各有特色？
- 对话是否推进剧情而非纯废话？

### 6. 角色塑造（10%）

| 分数 | 标准 |
|------|------|
| 9-10 | 角色行为完全符合人设，有成长或转变 |
| 7-8 | 角色基本符合人设，偶有出戏 |
| 5-6 | 角色行为勉强合理，但缺乏深度 |
| 3-4 | 角色行为违背人设，显得突兀 |
| 0-2 | 角色塑造失败，像工具人 |

**自检问题**：
- 主角的每个决定是否符合其性格？
- 反派是否有足够的智商？还是纯粹送人头？
- 角色是否有独特的口头禅或习惯？

### 7. 字数达标（5%）

| 分数 | 标准 |
|------|------|
| 9-10 | 2500-3000 字，完美范围 |
| 7-8 | 2000-2500 或 3000-3500 字，可接受 |
| 5-6 | 1500-2000 或 3500-4000 字，略偏 |
| 3-4 | 1000-1500 或 4000-5000 字，需要调整 |
| 0-2 | 低于 1000 或超过 5000 字，严重不达标 |

### 8. 去AI味（10%）- 绝对零度写作

基于"绝对零度写作"原则评分。

| 分数 | 标准 |
|------|------|
| 9-10 | 完全符合绝对零度原则：无比喻、无直白心理、无禁语 |
| 7-8 | 基本符合，偶有轻微违规 |
| 5-6 | 有明显违规：存在比喻/直白心理/禁语，需修改 |
| 3-4 | 违规严重：多处比喻、直白心理、禁语表词汇 |
| 0-2 | 完全AI腔：满屏比喻、解释、心理描写 |

**绝对零度必检项**（详见 `anti-ai-detection.md`）：
- [ ] 无任何比喻词（像/如同/仿佛/宛如/犹如……）
- [ ] 无直白心理描写（他很XX/她感到XX）
- [ ] 无禁语表词汇（冷笑/颤抖/指节泛白/嘴角勾起……）
- [ ] 无"被"字结构滥用
- [ ] 无"很"/"非常"滥用
- [ ] 无"他动了"/"就是现在"等烂俗套路
- [ ] 段落不超过4行
- [ ] 无解释性文字，让读者自己理解

## 自动评分提示词

每次生成章节后，使用以下提示词进行自评：

```
请对第XX章进行质量评分，检查以下要点：

1. 【爽点】本章的爽点是什么？是否足够强烈？
2. 【连贯】角色状态、地点、等级是否与记忆文件一致？
3. 【节奏】是否遵循"铺垫→冲突→高潮→钩子"结构？
4. 【悬念】章末钩子是什么？够不够吸引人？
5. 【对话】各角色说话风格是否有区分？
6. 【字数】字数是否在 2000-3000 范围内？
7. 【去AI味】是否有"然而/但是/于是/紧接着"等过渡词？段落是否过长？是否有直白心理描写？

最终给出 0-10 的综合评分和简短评语。
```

## 质量改进触发条件

| 触发条件 | 处理方式 |
|---------|---------|
| 单章评分 < 6.0 | 标记为 D 级，需重写该章 |
| 连续 2 章评分 < 7.0 | 触发节奏分析，暂停生成检查问题 |
| 评分 < 7.5 的章节超过 30% | 暂停创作，分析 ERROR.md，调整提示词 |
| 某维度连续 3 章 < 5.0 | 针对该维度优化（如：对话质量持续差） |
| 去AI味评分 < 5.0 | 必须进行去AI味修改，使用 anti-ai-detection.md 指南 |
| 连续 3 章去AI味 < 7.0 | 重写质量评分系统中的语言风格部分 |
