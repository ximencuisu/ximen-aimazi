# 作家风格参考库

用于在 `ximen-aimazi` 中做文风定调、节奏借鉴和题材嫁接。

这份文档的目标不是“模仿谁的句子”，而是拆出可迁移的写法：

- 节奏怎么推
- 爽点怎么埋
- 世界观怎么立
- 反派怎么压
- 氛围怎么造

---

## 使用原则

1. 一次优先选择 `1 位主参考 + 1 位辅参考`，超过 2 位容易风格打架。
2. 只借结构、节奏、冲突、悬念、人物关系处理，不逐句模仿。
3. 先确定基础文风（A-E），再叠加作家技法参考。
4. 风格参考要写入 `assets/STYLE-TEMPLATE.md` 的 `ACTIVE_AUTHOR_STYLE`。

## 重点作家技法拆解

### 天蚕土豆

- 关键词：打脸节奏、升级清晰、期待感管理
- 适合场景：退婚流、逆袭流、标准玄幻爽文
- 可迁移技法：
  - 前期压低主角处境，快速给第一次翻盘
  - 每次升级都立刻兑现战斗或身份收益
  - 阶段反派一层比一层高，不让主角空转
- 使用提醒：不要只学“嘲讽反派”，更要学“每次变强都改变局势”

### 我吃西红柿

- 关键词：流畅升级、长期成长、稳定连载感
- 适合场景：修仙、吞噬、成长型长篇
- 可迁移技法：
  - 用清晰目标驱动下一阶段修炼或冒险
  - 小闭环不断，减少空章节
  - 战斗与成长绑定，成长不只是数字变化
- 使用提醒：避免只剩流水账升级，要给阶段性代价和取舍

### 辰东

- 关键词：史诗感、远古秘辛、世界层层抬升
- 适合场景：大世界玄幻、神魔史诗、群像成长
- 可迁移技法：
  - 早期埋远期谜团，让后期有“世界突然变大”的惊喜
  - 关键大战前先做情绪抬升和宿命感铺垫
  - 重要对手要有分量，不是单纯送经验包
- 使用提醒：大设定要回收，不然容易只剩空喊史诗

### 爱潜水的乌贼

- 关键词：类型融合、设定严密、信息递进
- 适合场景：蒸汽朋克、诡秘、体系复杂的类型文
- 可迁移技法：
  - 先建立规则，再放反转
  - 用组织、派系、路径制造探索纵深
  - 每一层新信息都改写读者对旧信息的理解
- 使用提醒：复杂设定必须服务剧情，不要为炫设定牺牲可读性

### 滚开

- 关键词：压迫感、危险感、超展开、硬推进
- 适合场景：克系、异化、黑暗成长、极端世界
- 可迁移技法：
  - 用高密度异常细节制造不安
  - 让主角持续面对更大层级的危险
  - 关键节点果断转阶段，不拖泥带水
- 使用提醒：压抑氛围要配破局快感，否则读者容易疲劳

### 黑山老鬼

- 关键词：诡异、人性、秩序崩坏后的情绪张力
- 适合场景：悬疑、灵异、怪谈、末世异变
- 可迁移技法：
  - 先写“日常微妙不对劲”，再逐渐拉开恐怖面
  - 反派和怪物都要有其规则与逻辑
  - 角色选择要暴露人性，不只推动剧情
- 使用提醒：不要只堆氛围词，要让恐惧来自具体后果

### 佛前献花

- 关键词：规则、代价、解法、压缩叙事
- 适合场景：规则流、怪谈、灵异生存
- 可迁移技法：
  - 每个诡异现象先给限制，再给代价，再留破解空间
  - 主角强不等于无敌，而是更懂规则
  - 战斗写成博弈，而不是纯火力对轰
- 使用提醒：规则必须稳定，不能为推进剧情临时改规则

### 狐尾的笔

- 关键词：点子密度、本土化克系、癫狂推进
- 适合场景：高概念题材、认知混乱、反常识世界
- 可迁移技法：
  - 一个卷只抓一个最核心的怪点，把它挖透
  - 让设定从“奇”转成“必然”
  - 节奏推进时少解释，多让角色撞上后果
- 使用提醒：高概念越强，叙事越要清楚

### 老鹰吃小鸡

- 关键词：长篇驾驭、热血、多线推进、烟火气
- 适合场景：高武、都市异能、超长连载
- 可迁移技法：
  - 主线升级之外持续保留同伴、阵营、时代变化
  - 高潮前先把情绪、人情债和立场冲突攒够
  - 让强者也有生活感，不全是工具性战力单位
- 使用提醒：长篇不是每章都炸，而是高潮之间也要有热度

### 轻泉流响

- 关键词：轻松、治愈、羁绊、陪伴感
- 适合场景：御兽、轻小说、慢热成长
- 可迁移技法：
  - 用互动而不是旁白证明关系升温
  - 把训练、日常、成长做成期待点
  - 小目标也要有即时满足
- 使用提醒：轻松不等于没主线，仍要有阶段推进

### 最白的乌鸦

- 关键词：幽默反套路、设定碰撞、轻快节奏
- 适合场景：搞笑修仙、法治修仙、规则喜剧
- 可迁移技法：
  - 严肃设定和现实逻辑硬碰，制造反差
  - 对话里多用立场冲突，不靠段子堆笑点
  - 主角聪明但不油腻
- 使用提醒：反套路也要回到剧情推进，不能只图抖机灵

## 落地写法

当用户指定风格时，建议输出以下结构：

```markdown
## 本次作家技法参考
- 主参考：XXX
- 辅参考：YYY

## 提炼的可迁移技法
- 节奏：
- 世界观：
- 对话：
- 情绪：
- 章末钩子：

## 禁止误用
- 不逐句模仿
- 不照搬代表桥段
- 不复制专属名词体系
```

写完后，将结果同步到 `assets/STYLE-TEMPLATE.md`。
