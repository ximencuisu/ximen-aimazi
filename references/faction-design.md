# 势力设计指南

> 何时加载：Phase 2 世界观构建、Phase 3 角色塑造（涉及多势力冲突时）

势力（门派/家族/国家/组织/商会/联盟）是长篇网文世界观的骨架。好的势力设计能让主角"借势—破势"的弧线有处可借，让冲突有根、让升级有阶梯、让伏笔有容器。本指南给出从层级、利益、关系、节奏到主角互动的五层设计方法。

---

## 一、势力层级设计

长篇至少要有"四层势力金字塔"，让主角在每个阶段都有"够得着—够不着"的梯度感。单层势力会让中后期无势可借、无势可破。

### 四层金字塔

| 层级 | 规模特征 | 主角关系 | 出场卷次 | 设计要点 |
|------|----------|----------|----------|----------|
| **小势力** | 一个家族/一个小门派/一个商队/一个街头帮派 | 起步期的同行/对手/靠山 | 第 1-2 卷 | 必须有具体面孔（掌门/家主），不能是抽象组织；主角在此学到规则 |
| **中等势力** | 一个区域级门派/一个郡级家族/一个城邦商会 | 中段盟友或对手 | 第 2-4 卷 | 内部要有派系分裂（鹰派/鸽派），给主角介入缝隙 |
| **大势力** | 一国/一洲级宗门/一个帝国/一个跨域商会 | 后期靠山或终极对手 | 第 3 卷起逐步登场 | 设"执掌者会议"型决策层，主角需要打交道的是代表而非整体 |
| **隐世势力** | 上古遗族/禁地主人/沉睡的神祇/远古审判者 | 揭秘卷/高潮卷的变量 | 第 4 卷后伏笔、第 5+ 卷登场 | 出场前必须有 2-3 处伏笔（古籍/传说/遗迹），避免机械降神 |

### 跨度设计要点

- **地理跨度**：每层势力的影响半径要差一个量级（小势力一城、中等一郡、大势力一洲、隐世跨界），避免"满地图都是同级对手"的扁平感。
- **时间跨度**：隐世势力要有"千年不出，一出惊天"的时间感，与大势力"百年经营"形成对比。给隐世势力一个"苏醒条件"，作为长线悬念。
- **力量跨度**：相邻层级之间的实力差要让主角"靠智谋/借势能赢，正面硬刚必输"，而不是平推。

### 实操要点

1. 开书时先把四层各写 1-2 个势力名 + 一句话定位，放进 `STORY_BIBLE.md`，哪怕中后期才用。
2. 每个势力必须有"一个核心资源/秘技/地盘"，这是它存在的理由，也是主角借势/破势的抓手。
3. 同层势力之间要有竞争关系（争地盘/争资源/争正统），主角才能左右逢源。

### 反面案例

> ❌ 全书只有一个"魔教"当反派，主角从第 10 章打到第 300 章都是同一拨人，只是换了马甲（护法→长老→教主）。读者疲劳，升级无梯度。

### 正面案例

> ✅ 武侠文：小势力（镇上镖局）→ 中等势力（州府武林盟）→ 大势力（朝廷锦衣卫 + 江湖六大派）→ 隐世势力（藏经阁守墓人）。主角在每个层级都有不同的"借势"方式：镖局学规矩、武林盟借名望、锦衣卫借情报、守墓人借传承。

## 二、势力利益矩阵

势力不是为人设服务的背景板，而是为利益驱动的实体。给每个势力填一张"利益矩阵"，冲突会自然生长出来，不用硬编。

### 利益类型表

| 利益类型 | 说明 | 典型冲突触发 |
|----------|------|--------------|
| **生存利益** | 不被灭门/不被吞并/保住根基 | 外敌压境、资源枯竭 |
| **资源利益** | 矿脉/灵草/秘境/商路/人口 | 资源点刷新、新矿发现 |
| **名分利益** | 正统地位/盟主之位/朝廷册封 | 老盟主退位、新朝建立 |
| **传承利益** | 功法/血脉/神器/秘典归属 | 传承现世、传人争夺 |
| **扩张利益** | 吞并邻地/收编小势力/开辟新市场 | 实力此消彼长 |
| **复仇利益** | 历史血仇/灭门之恨 | 仇家后代现身、旧账翻出 |
| **信仰利益** | 教义正统/神祇归属/圣物守护 | 异端出现、圣物现世 |

### 利益矩阵模板（每个势力填一张）

```
势力名：______
核心利益（不可让步）：______        ← 这是它的底线，触碰即开战
重要利益（可以谈判）：______        ← 交换筹码
次要利益（可弃）：______            ← 让步用来示好
潜在利益（尚在谋划）：______        ← 伏笔来源
敌对利益方：______                  ← 它和谁有结构性冲突
盟友利益方：______                  ← 它靠谁
```

### 利益冲突识别

把两个势力的"核心利益"放一起，如果存在以下三种关系之一，冲突就必然发生：

1. **零和冲突**：同一资源只能一方占有（如唯一矿脉、唯一传承）。
2. **路径冲突**：双方扩张方向撞在一起（如都往同一座城发展）。
3. **身份冲突**：双方都自称正统，名分不两立（如两个"正宗"）。

如果两个势力之间没有结构性冲突，就不要硬让它们打架——让它们保持中立或交易关系，反而能给主角"借势"的空间。

### 实操要点

- 填矩阵时问自己："如果这个势力明天消失，谁会受益、谁会受损？"——受益方就是潜在加害者。
- 主角介入冲突的最佳位置是"次要利益冲突区"：赢了有收益、输了不至于死、还能两边要价。

## 三、势力关系网

四类基础关系 + 动态变化规则。关系不是静态标签，而是会随剧情演变的活物。

### 四类基础关系

| 关系类型 | 表现 | 主角可利用点 |
|----------|------|--------------|
| **同盟** | 公开互助、联合对外、共享部分资源 | 借盟友的势、但要还人情；盟友遇险必须救，否则信誉崩盘 |
| **敌对** | 公开对抗、互相打压、断绝往来 | 借一方打另一方、做渔翁；但容易被两方都盯上 |
| **中立** | 不结盟不树敌、做生意、各取所需 | 最佳"借势"对象——出价高就能请动，但也要防它倒戈 |
| **暗中博弈** | 表面和平、暗中渗透/刺探/下绊子 | 最适合主角做局——揭穿一层就搅动全局 |

### 关系动态变化规则

关系不是写死的，要随剧情推进而变化。给每组关系标一个"演变触发器"：

1. **同盟→破裂**：盟友被灭门未救 / 利益分配不均 / 一方暗中背叛被发现。
2. **敌对→缓和**：出现共同外敌 / 一方衰落求和 / 联姻/质子。
3. **中立→选边**：被卷入大战不得不选 / 主角开出无法拒绝的条件。
4. **暗中博弈→公开敌对**：阴谋败露 / 第三方揭穿 / 矛盾激化到压不住。

每卷至少要有 1-2 组关系发生演变，否则势力网会变死板。

### 关系网可视化

建议在 `STORY_BIBLE.md` 用一张文字版关系图记录：

```
        同盟
  [大势力A]──────[大势力B]
     │ 敌对          │ 暗中博弈
     │               │
  [中等势力C]      [中等势力D]
     │ 中立          │ 同盟
     │               │
  [小势力E]       [小势力F]
```

主角当前与哪几个势力有交集、是哪种关系，单列一行标注。

### 反面案例

> ❌ 全书势力关系一成不变：A 派和 B 派从头打到尾，C 派一直中立到完结。读者能预测每场冲突的阵营，毫无悬念。

### 正面案例

> ✅ A 派与 B 派第一卷死敌，第二卷因魔教压境被迫联手，第三卷魔教退去后因分赃不均再度反目，第四卷 A 派内乱被 B 派吞并。关系三次演变，每次都带出新冲突。

## 四、势力节奏

长篇每卷的势力格局都要有"破—立"的节奏变化，让世界在主角行动下被持续重塑。

### 每卷势力格局变化模板

| 卷次 | 势力格局动作 | 主角位置 |
|------|--------------|----------|
| 第 1 卷（起步） | 介绍小势力格局，1-2 个小势力被吞并/覆灭 | 旁观→参与→小有名气 |
| 第 2 卷（上升） | 中等势力登场，小势力洗牌 | 借中等势力之势崛起 |
| 第 3 卷（破局） | 大势力正式入场，旧格局被打破 | 在大势力夹缝中立足 |
| 第 4 卷（重塑） | 一方大势力倒台/分裂，新格局形成 | 成为新格局的关键变量 |
| 第 5 卷+（揭秘） | 隐世势力登场，颠覆表层格局 | 从"借势者"转向"造势者" |

### 借势与破势节奏

- **借势卷**：主角实力不足，靠投靠/结盟/挑拨借他人之势达成目标。本卷重点写"人情债"和"代价"。
- **破势卷**：主角实力到位，主动打破旧格局（灭一门、立一派、颠覆一盟）。本卷重点写"后果"和"新平衡"。

两种卷交替出现，避免一直借势显得窝囊，也避免一直破势显得空洞。

### 实操要点

1. 每卷开头问自己："本卷结束时，势力版图和卷初相比，多了什么、少了什么、变了什么？"——如果答案是"没变"，这卷的势力戏就是失败的。
2. 倒台的势力不要彻底消失，留一两个遗孤/残党/秘藏，作为后续卷的伏笔。
3. 新势力登场时，要带出一个"它为什么现在才出现"的理由，否则像凭空冒出。

## 五、与主角的"借势—破势"互动

主角和势力的关系是长篇的核心张力之一。分阶段设计互动策略，避免主角要么一直被势力碾压、要么一直碾压势力的极端。

### 四阶段互动策略

| 主角阶段 | 实力定位 | 互动策略 | 典型桥段 |
|----------|----------|----------|----------|
| **起步期** | 弱于小势力掌门 | **依附借势**：找个靠山、交投名状、当马前卒换取庇护 | 拜师、入赘、做任务换资源 |
| **上升期** | 持平中等势力中层 | **左右逢源**：在两方势力间做平衡，两边要价 | 当中间人、卖情报、挑拨离间 |
| **破局期** | 持平大势力执掌者 | **破立并举**：破旧联盟、立新秩序，自己成为势力核心 | 灭门、立派、夺盟主、颠覆王朝 |
| **超脱期** | 凌驾所有表层势力 | **揭幕隐世**：与隐世势力对话，重塑世界规则 | 入禁地、见古神、定新约 |

### 借势的代价设计

借势不能白借，每次借势都要留下"代价"，作为后续剧情的引子：

- **人情债**：欠下人情，日后被债主索还（往往是难办的事）。
- **名声债**：背上一个标签（某派弟子/某盟附庸），被敌对势力针对。
- **秘密债**：知道了势力不欲人知的秘密，被灭口或被要挟。
- **立场债**：表态站队后，失去其他势力的信任。

这些代价就是下一卷的剧情燃料。

### 破势的后果设计

破势之后世界不会自动太平，要写"破之后的乱"：

1. **权力真空**：旧势力倒了，地盘/资源/人口被多方争抢，新冲突爆发。
2. **连锁反应**：盟友变仇敌、仇敌变盟友，关系网大洗牌。
3. **更高层注意**：破了一个中等势力，引来了大势力关注（好或坏）。
4. **主角名声**：从"无名之辈"变成"风云人物"，出门被认出、被挑战。

### 反面案例

> ❌ 主角从头到尾单干，不结交任何势力，所有敌人都是一个人挑翻。中后期读者觉得世界是空的，没有"江湖感"。

### 正面案例

> ✅ 主角起步期拜入小门派学艺（借势），上升期被门派当弃子后投靠中等势力商会（转借势），破局期联合商会推翻压迫商会的州府武林盟（破势），超脱期发现武林盟背后是隐世遗族操控（揭幕）。四个阶段层层递进，每阶段都换了"势"的来源。

## 实操清单（写势力戏前过一遍）

- [ ] 四层势力金字塔是否都有具名势力？至少各 1 个。
- [ ] 每个登场势力是否填了利益矩阵？核心利益是否清晰？
- [ ] 本卷有没有至少 1 组势力关系发生演变？
- [ ] 主角本卷是"借势"还是"破势"？代价/后果是否埋好？
- [ ] 倒台势力是否留了遗孤/残党/秘藏作为伏笔？
- [ ] 新登场势力是否有"为什么现在才出现"的理由？
- [ ] `STORY_BIBLE.md` 的势力关系图是否同步更新？
