# 章首钩子与开篇技法

> 何时加载：Phase 6/8/10 章首钩子设计、开篇切入点与定调

本文件聚焦章首钩子（7 式）以及开篇相关的危机法则、定调层次、开头五要素、舞台搭建、场景节奏、飞卢风向、强噱头等技法。章尾钩子与悬念期待技法见 hook-ending.md。

## 章首钩子 7 式

### 1. 悬念对话开局
直接从一段意味深长的对话开始。
- "你确定要这么做？/ 确定。/ 那你就别后悔。"

### 2. 闪前碎片
先给一个结果碎片，再回到正常叙事。
- "后来他才知道，那个电话改变了所有事。但此刻他还一无所知。"

### 3. 倒计时开局
开头就建立紧迫感。
- "距离合同到期还有 72 小时。"

### 4. 神秘独白
第一人称的诡异内心独白。
- "我一直在想，那天如果我没有回头，一切会不会不一样。"

### 5. 反差场景
两个截然不同的场景并列。
- "一边是婚礼现场，香槟塔堆得老高。另一边，医院走廊的灯在闪。"

### 6. 未完成动作开局
动作进行中被截断。
- "他刚把钥匙插进锁孔——"

### 7. 意象预示
用环境意象暗示即将发生的事。
- "天边烧得通红，像是什么东西在着。她抬头看了一眼，继续走。"

## 开头危机五词法则

开头的危机/事件必须足够简单，让读者一眼看懂。检验标准：**用五个以内的词**向目标读者讲清楚事件是什么、到达什么状态，且不需要额外解释，读者就能完全明白甚至产生情绪。

### 判断标准

- 及格：五个词能说明"这是什么危机"+"为什么有这个危机"+"不解决会怎样"
- 不及格：需要大量额外解释读者才能理解 → 必然带来信息负担

### 为什么重要

读者的注意力有限，不喜欢思考，非常浮躁。开头如果事件复杂到需要解释，读者直接跑掉。日常常见事件天然有代入感（被辞退、被骗、被欺负），不需要解释就能产生情绪。

## 定调三层次与开篇信息控制

**定调三层次**：
- 第一层：情绪氛围（苦大仇深/轻松愉快/花前月下）
- 第二层：故事走向（种田/争霸/冒险/恋爱）
- 第三层：价值观倾向（男频/女频/宅味/土味）
- 让读者感受故事的气质，确认是否是自己期待的故事

**黄金三章 = 回应期待**：
- 书名和简介已经高度浓缩了冲突、悬念、卖点
- 黄金三章的核心作用是快速回应点进这本书的读者的期待
- 不要把噱头藏在很后面——读者等不了
- 开篇围绕书名和简介构建冲突，开篇点题
- 公式：令人不安的处境 + 急待提升的地位 + 充满希望的外挂 + 能被展望的美好未来

**开篇信息控制原则**：
- 开篇需要干净简洁，克制表达欲
- 通过简单动作（睁眼、自语、观察）聚焦当下处境
- 奥卡姆剃刀：如无必要勿增实体——设定需立马使用

## 开头五要素与切入点选择

**开头五要素（第一章必须交代）**：
- 谁 → 主角信息
- 在哪里 → 世界背景
- 有什么 → 金手指
- 因为什么 → 矛盾冲突
- 要做什么 → 主线方向

**开头五条铁律**：
1. **简单点**：忌云里雾里、打哑谜的楔子
2. **不能偏**：开头剧情必须符合主线核心卖点
3. **要快**：切入剧情的速度要快，起因部分要略写
4. **要爽**：五章之内没有震惊就是失败
5. **不能平**：没有冲突矛盾就是失败

**切入点选择法（从0到e）**：

一个剧情 = 起因（无爽点）→ 经过（波澜）→ 结果（装逼）→ 后续（收获/钩子）

| 切入方式 | 故事线 | 效果 |
|----------|--------|------|
| 0=穿越 | 顺写全部 | 最慢，最磨蹭 |
| a=准备参赛 | 顺写 | 慢，起因占太多篇幅 |
| b=海选现场即将登台 | 倒叙/起因后置 | 推荐，最常见的正确写法 |
| c=舞台上马上开唱 | 更接近结果 | 更快，强力推荐 |
| d=演唱中观众震惊 | 直接高潮 | 太急 |
| e=赛后获得奖励 | 错过经过 | 不推荐 |

**最佳切入点 = b 或 c**——保留经过和结果的完整展开空间。

**略写的艺术**：
- 起因部分必须简明扼要，千万不能事无巨细
- 一旦展开某个小环节就不得不一直展开，停不下来
- 参考模板：睁眼穿越 → 三两句回忆 → 当前处境（危机）→ 叮的一声系统 → 准备装逼
- 略写的极致：第一章仅用两三百字跳过所有起因

**矛盾冲突的广义理解**：
- 不一定需要敌人和反派，本质是"现实情况和预期需求的冲突"
- 危机困境型：被围杀/被羞辱/没钱付彩礼
- 隐形冲突型：主角想装逼但无法装逼 → 需要金手指了所以才有"叮"
- 宏观冲突型：国家军工落后亟需追赶

## 舞台搭建——大小与高度

**舞台 = 供主角表演的地方 + 台下有观众**。爽感差距来自两个维度：舞台的【大小】和【高度】。

**舞台大小（观众数量）**：
- 单排冲大师（无观众） vs 线上万人围观（有观众） vs 鸟巢八万现场+数百万网络观众（全民）
- 扩大方法：增加人物 → 增加目击者和见证者
- 设定室友/同学/同事从同一起点出发 → 主角一飞冲天形成"可悲的厚障壁"

**舞台高度（对手强度）**：
- 普通大师晋级赛 vs 王者局 vs S赛决赛面对历史第一战队
- 抬高方法：增加剧情 → 铺垫对手的强大、背负的使命、多重视托
- 多线铺垫叠加意义：被淘汰的队友托孤 + 不看好的父母到现场 + 全国观众的期待

**核心原则**：
- 在主角能装逼的时候 → 观众能拉多少就拉多少 → 意义能附加多少就附加多少
- 但前提是不崩 → 合理性底线不能丢
- 最忌讳白开水比赛：起床→去场馆→赢了→回来

## 场景目标驱动法与快慢节奏技巧

**场景目标 = 紧张感的核心**：
- 一句话陈述主角在这个场景中想达成的目标 → 尽早放在场景中
- 网文读者一目十行 → 用不同方式多次强调主角目标 → 让它印入读者脑海
- 示例："我必须找到妹妹，救她一命" / "我要杀掉怪物，否则就拼死一搏"

**增强紧张感四要素**：
1. **动机**：说明主角为何需要达成目标，失败将会有严重后果
2. **时间限制**：给主角设定实现目标的时间限制 → 紧张感更强烈
3. **障碍**：阻碍主角实现目标的障碍 → 必须比预想的更严重 → 不要过早透露细节
4. **递进危险**：以为脱险 → 凶手竟然藏在车里（出其不意）

**快节奏写作技巧**：
- 动作快速发生时使用 → 不要过多描写 → 少用形容词副词 → 短段落短句短词
- 短句传达喘息感和心跳加速 → 删掉所有多余词语 → 甚至使用不完整句子
- 例："忽的，她笑了" / "冰凉的双手，抓住了他的脖颈"

**慢节奏写作技巧**：
- 主角等待/无计可施时使用 → 中长段落 → 句子长度不一 → 调动多种感官
- 背景噪音描写 = 最有效的慢节奏技巧 → 漆黑中传来磨刀声 / 安静走廊门被缓缓推开
- 通过放缓节奏增强悬疑感 → 让读者比主角本人更焦虑

**快慢节奏的读者心理学**：
- 人类注意力以 15-20 分钟为周期 → 网文一章正好在这个范围
- 连续紧张后的舒缓期不能超过两章 → 超过两章读者就觉得"水了"
- 舒缓期的功能：消化上次紧张 → 铺垫下次紧张 → 角色互动加深（不是真的放松）

## 飞卢风向四要素

**风向一：期待感（最核心）**
- 简单纯粹爽的实用性降低了 → 拉慢故事期待感才是核心风向
- 警惕：故作神秘 ≠ 期待感 → 该解释的不解释 → 读者看不懂
- 期待感常用梗：曝光类/摊牌类/历史遗孤

**风向二：主线清晰**
- 前几年飞卢无主线只靠爽也能出成绩 → 现在概率低 N 倍
- 主线清晰 + 足够吸引 → 即使收藏不高 → 上架后稳步提升

**风向三：收获感**
- 狂暴式收获感 = 飞卢主流之一 → 签到/选择/自动升级/挂机
- 收获类型：物品收获 + 实力变强 + 感情收获
- 主角用不用奖励不一定 → 但让读者感到满满的收获感

**风向四：冲突感**
- 飞卢开头突出两点：期待感或冲突感
- 从书名就突出冲突 → "拆了我的XX还想让我XX"
- 冲突感 → 形成期待感 → 读者期待冲突爆发的那一刻

## 强噱头与反噬效应

### 强噱头的双刃剑
反噬风险：满足后读者弃书、后续内容无法匹配前期期待、完读率在满足点后断崖下跌。

### 正确使用强噱头
1. 延迟满足——不要过早满足大期待，用小爽点拉住读者
2. 大爽点放在关键节点——如10万字上架时
3. 一级结构层层递进——逐步抬高
4. 提前铺设下一个大期待——满足当前之前，新期待已到位
5. 小爽点不断——维持追读

核心：小爽点可以不断，大爽点要尽量少。

### 换地图不换期待
换地图/开新卷是期待最容易断裂的时刻：
- 提前铺设人脉关系网，角色作为"故事源"串联不同阶段
- 旧地图的未解决线索延伸到新地图
- 新地图的冲突在旧地图结尾就已经埋下伏笔
- 换地图后尽快建立新的"安全感"和新的期待锚点
