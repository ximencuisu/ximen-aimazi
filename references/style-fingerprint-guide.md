# 风格指纹比对与漂移控制指南

> 何时加载：Continuation_Mode 下 Phase 11 一致性审核和 Phase 12 润色阶段。

## 概述

风格指纹比对是续写流程中确保 AI 续写章节与原作风格一致的核心机制。通过9项可量化指标的偏离度计算，客观判断续写是否偏离原作风格，并在偏离过大时触发回拉或阻断。

---

## 九项指标详解与计算方法

### 1. 平均句长

**计算方法**：
- 以句号、问号、感叹号为分隔切分句子
- 计算每句字符数（不含标点）的平均值
- 排除对话中的纯对话句（避免对话拉低平均值）

**漂移阈值**：±20%

**示例**：
- 原作基线：18.5字
- 续写实测：22.1字
- 偏离度：(22.1-18.5)/18.5 = 19.5% → 未超阈值

### 2. 句长分布区间

**计算方法**：
- 短句：≤10字
- 中句：11-25字
- 长句：>25字
- 计算三类句子的比例

**漂移阈值**：±20%（每个区间独立计算）

**示例**：
- 原作基线：短30%/中50%/长20%
- 续写实测：短25%/中52%/长23%
- 长句偏离：(23-20)/20 = 15% → 未超阈值

### 3. 平均段长

**计算方法**：
- 以空行为分隔切分段落
- 计算每段行数的平均值

**漂移阈值**：±25%

**示例**：
- 原作基线：2.3行/段
- 续写实测：3.1行/段
- 偏离度：(3.1-2.3)/2.3 = 34.8% → 超阈值！

### 4. 对话占比

**计算方法**：
- 识别引号标记的对话行
- 对话行数 / 总行数

**漂移阈值**：±15%

**示例**：
- 原作基线：35%
- 续写实测：28%
- 偏离度：(28-35)/35 = -20% → 超阈值！

### 5. 高频意象前20

**计算方法**：
- 提取续写章节的名词性意象词频率
- 与原作前20高频意象计算重合度
- 重合度 = 交集数量 / 20

**漂移阈值**：重合度 ≥ 60%（即至少12个意象与原作重合）

**示例**：
- 原作高频意象：剑气、灵石、丹药、阵法、符箓、法宝、灵兽、洞府、宗门、秘境、天劫、道心、仙草、灵脉、神识、飞剑、护盾、灵力、元神、仙器
- 续写高频意象：剑气、灵石、丹药、阵法、符箓、法宝、灵兽、洞府、宗门、秘境、天劫、道心、仙草、灵脉、神识、飞剑、护盾、灵力、元婴、渡劫
- 重合度：18/20 = 90% → 远超阈值

### 6. 高频动词前20

**计算方法**：同高频意象，计算重合度

**漂移阈值**：±20%（以重合度计算，≥80%为合格）

### 7. 句式偏好

**计算方法**：
- 统计陈述句/疑问句/感叹句/倒装句的比例
- 每类独立计算偏离度

**漂移阈值**：±20%

### 8. 动作-对话-情绪比例

**计算方法**：
- 将每段文字分类为动作描写、对话、情绪/心理描写
- 计算三类比例

**漂移阈值**：±20%

### 9. 原作回避表达倾向

**计算方法**：
- 检查续写章节是否出现原作从未使用的表达方式
- 特别关注 `references/banned-words.md` 中原作未使用的禁用词

**漂移阈值**：0（不得出现原作回避的表达）

## 漂移阈值默认值汇总

| 指标 | 默认阈值 | 说明 |
|------|---------|------|
| 平均句长 | ±20% | 超出则句式节奏偏离 |
| 句长分布 | ±20% | 每区间独立计算 |
| 平均段长 | ±25% | 段落偏长最常见 |
| 对话占比 | ±15% | 对话密度对读感影响大 |
| 高频意象 | 重合度≥60% | 意象词反映题材风味 |
| 高频动词 | 重合度≥80% | 动作词反映战斗/互动风格 |
| 句式偏好 | ±20% | 每类独立计算 |
| 动作-对话-情绪 | ±20% | 三类独立计算 |
| 回避表达 | 0 | 不得出现 |

**阈值调整**：用户可在 `设定/风格指纹.md` 中自定义各项阈值。

## 阻断与回拉规则

### 阻断规则

当续写章节有 **3项以上** 指标超过漂移阈值时：
1. 阻断 Phase 12 润色阶段
2. 向用户展示风格对比报告
3. 用户选择：
   - **重写本章**：回到 Phase 10 重新生成
   - **调整阈值**：修改 `设定/风格指纹.md` 中的阈值
   - **接受偏离**：继续进入 Phase 12，但标记为"风格偏离已确认"

### 回拉规则（Phase 12 润色阶段执行）

针对每项超阈值指标的具体回拉操作：

| 超阈值指标 | 回拉操作 |
|-----------|---------|
| 平均句长偏高 | 拆分长句为短句，增加句号断句 |
| 平均句长偏低 | 合并过短的碎片句 |
| 平均段长偏高 | 在长段落中寻找自然断点拆分 |
| 对话占比偏低 | 将部分叙述转为对话形式 |
| 对话占比偏高 | 将部分对话转为动作描写 |
| 高频意象重合度低 | 替换非原作意象为原作高频意象 |
| 情绪描写占比偏高 | 将直白情绪描写转为动作暗示 |
| 出现回避表达 | 直接替换为原作常用表达或删除 |

## 与 Phase 11/12 的集成

### Phase 11 集成

1. 在完成原有的一致性审核后，额外执行风格指纹比对
2. 产出 `追踪/续写风格对比_第{N}章.md`
3. 如果3项以上超阈值 → 阻断进入 Phase 12

### Phase 12 集成

1. 在完成原有的去AI味检查和润色后，额外执行风格指纹回拉
2. 回拉后重新计算9项指标
3. 在风格对比报告中追加"润色后"列
4. 如果回拉后仍有3项以上超阈值 → 再次提示用户决定

## 常见问题

### Q：风格指纹一定要9项全算吗？

是的。9项指标互相补充，缺项可能导致风格偏离无法被检测到。如果某项指标无法计算（如对话为0的纯叙述章节），标注"不适用"并跳过该项的阈值判断。

### Q：漂移阈值可以调整吗？

可以。用户可在 `设定/风格指纹.md` 中自定义各项阈值。建议不要把阈值调得太宽松，否则风格比对失去意义。

### Q：原作本身风格不统一怎么办？

如果原作前中后期风格变化较大，建议分段提取风格指纹（如前30章/中30章/后30章分别提取），续写时使用对应段的指纹作为基线。

### Q：风格比对和去AI味检查冲突怎么办？

风格比对优先级高于去AI味。如果回拉风格需要使用原作常用但去AI味禁用的表达，以风格比对为准。但 `references/banned-words.md` 中的硬禁用词仍然不得使用。
