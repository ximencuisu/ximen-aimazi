## AI写作指纹识别

AI生成的文本有几类高频特征，逐项排查即可定位问题：

### 1. 比喻词泛滥

AI喜欢用比喻来"增色"，但网文不需要这些：

```
禁止：像、就像、好像、如同、仿佛、宛如、犹如、好似
```

```
❌ 他像箭一样冲出去
✅ 他冲了出去

❌ 她的眼睛如同星辰般明亮
✅ 她眼睛很亮
```

### 2. 弱化副词堆叠

每1000字超过3个以下词汇 = AI签名：

```
微微、淡淡、缓缓、轻轻、不禁、不由自主
```

### 3. 直白心理描写

AI习惯直接告诉读者角色在想什么、感受什么，而不是通过行为展示：

```
❌ 他很紧张。她很焦急。他感到愤怒。
✅ 他手心全是汗。她眼眶撑开了些。他没出声。
```

### 4. 意义膨胀

AI喜欢把小事说成大事：

```
❌ "意义深远" → 写具体后果
❌ "前所未有" → 给出对比参照
❌ "可谓" → 删掉
```

### 5. 万能结论

章节结尾出现以下句式 = AI入侵：

```
❌ "未来可期" → 用未解决的紧张感结尾
❌ "前途无量" → 删
❌ "充满希望" → 写具体的下一步动作
❌ "他终于明白了……" → 删
❌ "这一夜，注定无人入眠" → 用动作收束
❌ "他不知道的是，更大的风暴即将来临" → 删
```

### 6. 论文体结构

小说中出现以下开头句 = AI痕迹：

```
"不难看出""由此可见""事实上""综上所述"
"于是乎""与此同时""从而""因而""诚然"
```

### 7. 三连排比癖

AI喜欢把事情凑成三个以显"完整"：

```
❌ 这次突破带来了创新、进步、变革
✅ 这次突破带来了进步

❌ 他冷静、沉着、有条不紊
✅ 他很冷静
```

## 中文去欧化

### 拒绝"很"和"非常"

```
❌ 他很高。她很美。这很重要。
✅ 他身材高大。她容貌出众。这事关成败。
```

### 拒绝"被"字滥用

```
❌ 他被激怒了。他被打败了。
✅ 他火了。他输了。
```

### 拒绝弱动词+名词

```
❌ 作出决定、进行分析、进行处理
✅ 决定、分析、处理
```

### 拒绝翻译腔

```
❌ 名为……的……
❌ 取而代之的是……
❌ 令人……
❌ 最……之一
❌ 如此……以至于……
```

## 快速检查表

### 必检项

- [ ] 无比喻词（像/如同/仿佛/宛如）
- [ ] 无"被"字滥用
- [ ] 无直白心理（他很XX/她感到XX）
- [ ] 无禁语表词汇（见 [banned-words.md](banned-words.md)）
- [ ] 无"很"/"非常"滥用
- [ ] 段落不超过4行
- [ ] 章尾无总结/升华/预告

### 常见错误速查

| 错误 | 正确 |
|-----|------|
| 他很紧张 | 他手心全是汗 |
| 像一把刀 | 直接描述话语内容 |
| 他动了 | 具体写他做了什么 |
| 嘴角勾起 | 嘴角动了 |
| 空气凝固 | 没人说话 |
| 沉默了 | 过了几秒 |
| 不仅...而且... | ...和... |
| 希望这有帮助 | 删掉 |
