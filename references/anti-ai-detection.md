# 去AI味指南（绝对零度写作）

整合"绝对零度写作"原则与Wikipedia AI写作识别标准，彻底消除AI痕迹。

---

## 核心理念

### 绝对零度写作三要素

1. **直接呈现** - 只写可观测事实，不写内心活动
2. **禁止修辞** - 无条件禁止比喻、隐喻、类比、拟人
3. **精确细节** - 重要细节精确到厘米/秒级

### 避免空洞写作

仅仅删除AI模式不够，还需要**注入灵魂**：
- 每句话长度和结构不要相同
- 有观点，不只是中性报道
- 承认不确定性和复杂情绪
- 适当使用第一人称
- 有幽默感、棱角、个性

## 第一章：AI高频词汇（零容忍）

### 高频AI词汇表

| 禁用词 | 替代方案 |
|--------|---------|
| **此外、而且** (Additionally) | 然后、还有、并且 |
| **关键、核心** (crucial/pivotal/key) | 直接删除或用具体词 |
| **体现、展示** (showcase/demonstrate) | 直接描述动作 |
| **强调、突出** (emphasize/highlight) | 直接描述 |
| **交织、交织** (interplay) | 删除或用具体词 |
| **重要** (significant/vital) | 直接删除 |
| **生动、活力** (vibrant) | 删除或用具体颜色 |
| **画卷、织锦** (tapestry) | 直接描述 |
| **证明、体现** (testament) | 直接描述 |
| **强调** (underscore) | 直接删除 |
| **深远** (enduring) | 删除或用具体时间 |
| **培育、促进** (foster/cultivate) | 直接描述动作 |
| **获得、赢得** (garner) | 直接用"获得" |
| **深入探讨** (delve into) | 直接用"研究" |

### Copula Avoidance（避免使用is/are）

AI喜欢用"serves as/boasts/represents"代替"is/has"：

```markdown
# 错误
林凡的炼丹系统 serving as 他成功的基石。
丹药拥有 87% 的成功率。

# 正确
林凡的炼丹系统是他成功的基石。
这炉丹药的成功率是87%。
```

## 第二章：比喻类比禁令

### 绝对禁止的比喻词

```
像、就像、好像、如同、仿佛、宛如、犹如、好似
```

### 禁止句式

```markdown
# 禁止
他像箭一样冲出去。
她的眼睛如同星辰般明亮。

# 正确
他冲了出去。
她眼睛很亮。
```

### 禁止声音/话语比喻

```markdown
# 禁止
他的话像一把锋利的刀，剖开了她的防线。

# 正确
他的话驳倒了她的所有论点。
```

## 第四章：语法和句式模式

### 1. Negative Parallelisms（否定排比）

```markdown
# 禁止
这不仅仅是关于成功，这是关于成长。
这不只是战斗，更是突破。

# 正确
这关乎成长。
这是战斗，也是突破。
```

### 2. Rule of Three（三项并列）

AI喜欢把观点强行凑成三项：

```markdown
# 禁止
这次突破带来了创新、进步、变革。
他冷静、沉着、有条不紊。

# 正确
这次突破带来了进步。
他很冷静。
```

### 3. Elegant Variation（同意词替换）

AI会过度使用同义词避免重复：

```markdown
# 禁止
林凡 face 了挑战。主角 encounter 了困难。
英雄 overcome 了障碍。

# 正确
林凡 face 了挑战。主角也 face 了困难。
英雄 overcome 了这些障碍。
```

### 4. False Ranges（虚假范围）

```markdown
# 禁止
从宇宙大爆炸到黑洞，从星辰到暗物质。

# 正确
书中讨论了大爆炸、恒星形成和暗物质理论。
```

## 第五章：中文去欧化

### 拒绝滥用"很"和"非常"

```markdown
# 禁止
他很高。她很美。这很重要。

# 正确
他身材高大。她容貌出众。这事关成败。
```

### 拒绝被动"被"字

```markdown
# 禁止
他被激怒了。他被打败了。

# 正确
他火了。他输了。
```

### 拒绝弱动词+名词

```markdown
# 禁止
作出决定、进行分析、进行处理

# 正确
决定、分析、处理
```

### 拒绝虚词

```markdown
# 禁止
残忍地杀害、小心翼翼地拿起

# 正确
残杀、拿起
```

### 拒绝翻译腔

```
名为……的……
取而代之的是……
令人……
最……之一
如此……以至于……
```

## 第六章：风格问题

### Em Dash滥用

```markdown
# 禁止
林凡站了出来——所有人都看向他——
他开口了——声音在寂静中格外清晰。

# 正确
林凡站了出来。所有人的目光转向他。
他开口了。
```

### 粗体滥用

```markdown
# 禁止
他的**炼丹系统**检测到了药材。
他**运起灵气**，准备炼丹。

# 正确
他的炼丹系统检测到了药材。
他运起灵气，准备炼丹。
```

### 数字列表式标题

```markdown
# 禁止
## 第一步：准备工作
## 第二步：开始炼丹

# 正确
## 准备工作
## 开始炼丹
```

## 第七章：交流模式

### 禁用语气词

```
希望这有帮助、当然、绝对正确、
您想要……、请告诉我
```

### 禁用免责声明

```
截至目前、根据最新信息、
虽然具体细节有限、基于可用信息
```

### 禁用谄媚语气

```markdown
# 禁止
好问题！您完全正确！这是个绝佳的观点！

# 正确
（直接回答问题）
```

## 第八章：填充词和Hedging

### 冗余短语

```markdown
# 禁止 → 正确
为了实现这个目标 → 为了
因为事实上下雨了 → 因为下雨了
在这个时间点 → 现在
在需要帮助的时候 → 需要帮助时
系统有能力处理 → 系统能处理
重要的是注意到数据显示 → 数据显示
```

### 过度Hedging

```markdown
# 禁止
这可能潜在地表明该政策或许会对结果产生影响。

# 正确
该政策可能影响结果。
```

### 空洞结尾

```markdown
# 禁止
未来充满希望。激动人心的时刻即将到来。
这是一个正确的方向。

# 正确
公司计划明年再开两家店。
```

## 第九章：直接呈现原则

### 禁止直白心理

```markdown
# 禁止
他很紧张。她很焦急。他感到愤怒。

# 正确（通过动作展示）
他手心全是汗。她眼眶撑开了些。他没出声。
```

### 禁止臆测含义

```markdown
# 禁止
他伸出手，似乎想触碰她，但怕惊扰她而缩回。

# 正确
他向她的后背伸出手，手在半空中停住，没触碰她就缩回。
```

### 禁止解释性文字

让读者自己理解，不告诉他们"这意味着什么"。

## 第十章：场景改写对照

### 打脸场景

```markdown
# AI写法
林凡淡淡地扫了赵天罡一眼，从容地取出那枚二阶凝元丹。
丹药表面流转着淡金色光泽，药香浓郁得让人心醉。
赵天罡脸色大变，眼中满是不可置信。
全场一片死寂，所有人都惊呆了。

# 绝对零度
林凡把丹药往桌上一扔。
"二阶。"
没人说话。
赵天罡的脸抽了一下。
```

### 战斗场景

```markdown
# AI写法
林凡运起体内灵气，灵力在经脉中快速运转。
他的拳头泛起淡淡的金光，如同一轮小太阳。
然而赵天罡的速度太快了，他的护体灵气瞬间被击碎。

# 绝对零度
林凡一拳轰出去。
赵天罡的护体灵气碎了。
他还愣着。
```

### 情绪暗示

```markdown
# AI写法
他心里很紧张，因为他知道这次炼丹的成功率很低。

# 绝对零度
林凡握着灵草，手心全是汗。
```
