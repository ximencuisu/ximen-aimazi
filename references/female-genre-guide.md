# 第一部分：题材与节奏

## 女频核心情绪

女频读者追求的核心情绪体验，可归纳为五大类：

1. **安全感** — 被保护、被守护、有依靠，不再独自面对风雨
2. **被偏爱** — 在所有人中唯独选我，偏心偏到不讲道理
3. **逆袭** — 从低谷爬起来，让曾经看不起我的人仰望
4. **独立** — 不依附任何人，靠自己的能力活出精彩
5. **被选择** — 不是将就，不是替代，而是坚定地被选择

> 写作要点：每本女频小说至少锚定1-2个核心情绪，所有情节设计围绕核心情绪展开。核心情绪决定题材归属和读者预期。

## 女频题材框架

### 甜宠文

- **核心情绪**：被偏爱
- **结构**：误会→解释→更甜（螺旋上升，每次误会都是升温契机）
- **爽点**：独宠/护短/当众表白/醋意大发/霸道宣布主权
- **节奏**：日常甜度密度高，小误会不超过3章必须解开
- **关键技法**：
  - 男主对女主的偏爱要有"排他性"——对别人冷若冰霜，对女主温柔如水
  - 误会不能是原则性问题，否则读者会站不住
  - 每个配角的存在都是为了衬托CP甜度或制造小波折
- **常见雷区**：误会拖太久、男主人设崩塌（对别人也温柔）、女主太作

### 虐恋文

- **核心情绪**：痛并期待
- **结构**：甜蜜→误会/背叛→虐心→和好/BE
- **爽点**：追妻火葬场/后悔/挽回/泪目告白
- **节奏**：前30%建立甜蜜基础→中间40%虐→后30%和好或BE
- **关键技法**：
  - 虐的前提是甜——没有甜过就没有虐的痛感
  - 误会/背叛必须有合理逻辑，不能强行降智
  - 追妻火葬场阶段要让男主真正痛苦，读者才能解气
  - BE要虐得有美感，让读者痛但觉得值
- **常见雷区**：为虐而虐、男主渣得没底线、女主圣母原谅太快

### 宫斗/宅斗

- **核心情绪**：智谋碾压
- **结构**：低位→结盟→算计→翻盘
- **爽点**：打脸绿茶/反杀/上位/以牙还牙/借刀杀人
- **节奏**：每5-8章一个小翻盘，每卷一个大翻盘
- **关键技法**：
  - 女主的智谋要"合理"——信息差、人脉差、资源差，不是开挂
  - 反派不能太蠢，否则赢了也没成就感
  - 每次翻盘都要有铺垫，不能突然天降神兵
  - 宅斗侧重家族内部关系，宫斗侧重权力博弈
- **常见雷区**：反派降智、女主全知全能、配角工具人化

### 种田/经商

- **核心情绪**：独立成长
- **结构**：贫困→小生意→扩大→危机→解决
- **爽点**：赚钱/被认可/改变命运/带动身边人一起好
- **节奏**：慢热型，日常占比大，每10章一个阶段性成果
- **关键技法**：
  - 经营过程要写细——进货、定价、竞争、口碑，读者爱看过程
  - 危机不能太假，要符合时代和行业逻辑
  - 感情线是调味品不是主菜，但要有
  - 配角群像要丰富——邻里、同行、贵人、小人
- **常见雷区**：赚钱太容易、金手指太夸张、感情线喧宾夺主

### 重生文

- **核心情绪**：改命
- **结构**：前世惨→重生→逐层改命
- **爽点**：预知/避坑/打脸前世仇人/改写命运
- **节奏**：前世回忆穿插，每改一个命就有对应爽点
- **关键技法**：
  - 前世要惨得让人心疼，但篇幅不能太长（10%以内）
  - 重生后不能全知全能——知道大方向但细节要重新摸索
  - 每个改命节点都要有前世对比，形成爽感
  - 逐步改命而非一步到位，保持期待感
- **常见雷区**：前世写太多、重生后太顺、忘了前世设定

### 快穿/穿书

- **核心情绪**：多世界体验
- **结构**：任务→完成→新世界
- **爽点**：每个世界不同CP/身份/玩法/反转
- **节奏**：每个世界10-30章，快节奏、高密度
- **关键技法**：
  - 每个世界要有明确的核心梗，不能千篇一律
  - 世界之间的过渡要自然，可以用系统/主神空间串联
  - 每个世界的女主可以有不同的性格侧重
  - 快穿要"快"，不能在一个世界磨太久
- **常见雷区**：世界同质化、任务太机械、缺乏情感投入

## 女频爽点设计

### 高频爽点清单

| 爽点类型 | 具体表现 | 适用题材 |
|---------|---------|---------|
| 打脸绿茶 | 当众揭穿/反将一军/让对方自食其果 | 宫斗/宅斗/重生 |
| 宠妻 | 当众护短/霸道宣布主权/为女主破例 | 甜宠/虐恋 |
| 护短 | 我的人谁都不能欺负/替女主出头 | 甜宠/古言 |
| 独宠 | 后宫三千只取一瓢/对别人冷对女主暖 | 甜宠/宫斗 |
| 身份曝光 | 原来她是XX/隐藏身份震惊全场 | 重生/穿书/古言 |
| 逆袭 | 从被看不起到让人仰望 | 种田/经商/重生 |
| 被选择 | 不是替代品/坚定地选择你 | 甜宠/虐恋 |
| 守护 | 默默保护/替你挡刀/为你牺牲 | 虐恋/仙侠 |

### 爽点叠加法则

- 单一爽点力度不够，至少叠加2-3个
- 例：身份曝光 + 打脸绿茶 + 宠妻 = 爽感翻倍
- 爽点要有铺垫和延迟，不能随叫随到
- 大爽点前必须有压抑，压抑越深释放越爽

## 女频节奏特征

### 与男频节奏的核心差异

1. **感情线密度高于男频** — 感情线是主线不是副线，每章都要有感情推进
2. **日常互动占比大** — 吃饭、逛街、聊天、小打小闹都是有效内容
3. **情绪波动更细腻** — 不是大起大落，而是细水长流中的涟漪
4. **冲突多来自人际关系** — 婆媳、闺蜜、情敌、家族，而非打怪升级

### 女频节奏模板

```
[日常甜] → [小波折] → [解决+更甜] → [日常甜] → [大波折] → [虐] → [解决+大甜]
```

- 日常甜的篇幅约占60%
- 小波折3-5章一个，大波折10-15章一个
- 每次波折解决后甜度要升级，不能回到原点
- 全书不能超过3个大波折，否则读者疲劳

### 情绪曲线设计

- 女频的情绪曲线是"螺旋上升"而非"锯齿形"
- 每个低谷都是为下一个高峰蓄力
- 情绪不能断崖式下跌，要有过渡
- 高潮后的回落要温柔，给读者喘息空间

## 女频与男频的核心差异对比表

| 维度 | 女频 | 男频 |
|------|------|------|
| 核心驱动力 | 情感/关系 | 实力/地位 |
| 主线 | 感情线 | 升级线 |
| 爽点来源 | 被爱/被选/逆袭/智谋 | 升级/打脸/装逼/碾压 |
| 冲突类型 | 人际关系/情感纠葛 | 实力对抗/利益争夺 |
| 日常占比 | 高（60%+） | 低（30%左右） |
| 节奏 | 螺旋上升，细水长流 | 阶梯式跳跃，快节奏 |
| 读者代入 | 情感代入（我想被这样爱） | 力量代入（我想这么强） |
| 金手指 | 多为信息差/预知/系统 | 多为实力增强/特殊能力 |
| 反派类型 | 绿茶/白月光/婆婆/情敌 | 霸道少主/宗门/势力 |
| 高潮设计 | 表白/和好/身份曝光/当众护短 | 击败强敌/突破境界/震惊全场 |
| 结局偏好 | HE为主，BE要有美感 | 必须赢，开放式都少 |
| 字数偏好 | 中篇为主（50-150万） | 长篇为主（200万+） |
| 更新节奏 | 稳定日更即可 | 爆更效果更好 |
| 开篇重点 | 情感共鸣+人设 | 冲突+金手指 |
| 读者容忍度 | 对逻辑漏洞较宽容 | 对逻辑漏洞零容忍 |
| 最不能忍 | 男主渣/女主作/强行虐 | 主角怂/节奏慢/水字数 |

# 第二部分：角色设计

## 女主类型

### 白莲花

- **核心特征**：外表柔弱内心坚韧，看似无害实则有底线
- **适合题材**：虐文/宫斗/宅斗
- **人设要点**：
  - 柔弱是表象，坚韧是内核——关键时刻绝不退缩
  - 善良但不圣母，有原则有底线
  - 成长弧线明显：从隐忍到反击
- **爽点来源**：以柔克刚/弱者逆袭/隐忍后的爆发
- **写作禁忌**：不能真傻真弱，否则读者会弃文；不能圣母到没底线

### 黑莲花

- **核心特征**：表面无害实则心机深，善于伪装和借势
- **适合题材**：宅斗/宫斗/重生
- **人设要点**：
  - 外表人畜无害，内心算无遗策
  - 善于利用别人的弱点，但有自己的底线
  - 不是纯粹的恶，而是被环境逼出的生存智慧
- **爽点来源**：扮猪吃虎/反杀/算计得逞/真面目曝光
- **写作禁忌**：不能黑得没底线，要有让人共情的理由；算计不能太刻意

### 女强型

- **核心特征**：实力碾压不靠男人，独立自主有主见
- **适合题材**：玄幻/都市/经商
- **人设要点**：
  - 实力是自己的，不是男人给的
  - 可以有感情线，但不能依附于男人
  - 强大但有软肋，不是无情机器
- **爽点来源**：实力碾压/独立解决危机/不需要拯救
- **写作禁忌**：不能强到没朋友、没弱点；感情线不能太弱也不能喧宾夺主

### 甜软型

- **核心特征**：可爱治愈系，让人想保护
- **适合题材**：甜宠/日常/校园
- **人设要点**：
  - 软萌但不蠢，可爱但有主见
  - 善于发现生活中的小确幸
  - 关键时刻有自己的坚持
- **爽点来源**：被宠/被偏爱/小机灵/反差萌
- **写作禁忌**：不能蠢到让人生气；不能作到让读者想打人

### 清冷型

- **核心特征**：高岭之花，不食人间烟火
- **适合题材**：仙侠/古言/玄幻
- **人设要点**：
  - 外表清冷内心有温度，反差是关键
  - 不轻易动情，一旦动情就是全部
  - 强大但不张扬，低调但有实力
- **爽点来源**：动情反差/为一人破例/冷面下的温柔
- **写作禁忌**：不能冷到没感情；动情不能太快太突然

## 男主类型

### 霸总

- **核心特征**：强势占有欲，说一不二
- **适合题材**：甜宠标配
- **人设要点**：
  - 对外冷酷霸道，对女主温柔独宠
  - 占有欲强但不能控制欲变态
  - 强大是基础，偏爱是灵魂
- **经典台词风格**："你是我的""谁准你看别人""我说不行就是不行"
- **写作禁忌**：不能霸道到不尊重女主；占有欲和控制欲要有边界

### 暖男

- **核心特征**：温柔体贴，润物无声
- **适合题材**：治愈系/校园/日常
- **人设要点**：
  - 温柔不是软弱，是有力量的包容
  - 默默守护，关键时刻挺身而出
  - 细节见真心——记住女主的每个小习惯
- **经典台词风格**："我一直在""没关系，有我""你开心就好"
- **写作禁忌**：不能暖到没主见；不能对所有人都一样暖（要偏心）

### 病娇

- **核心特征**：偏执占有，爱到扭曲
- **适合题材**：虐恋向/暗黑系
- **人设要点**：
  - 爱是执念，得不到就毁灭
  - 表面正常，内心疯狂
  - 对女主的占有欲超出常理
- **经典台词风格**："你只能是我的""想逃？试试看""我不会让你离开"
- **写作禁忌**：不能只有病没有爱；要有让人心疼的过去；行为要有底线

### 腹黑

- **核心特征**：表面温和实则算计，运筹帷幄
- **适合题材**：宅斗/宫斗/权谋
- **人设要点**：
  - 笑着把人算计了，对方还感恩戴德
  - 对女主的算计是保护不是伤害
  - 智商在线，算无遗策
- **经典台词风格**："我怎么会害你呢""放心，都安排好了""有趣"
- **写作禁忌**：不能算计女主（除非是为了保护）；不能聪明到没对手

### 高岭之花

- **核心特征**：清冷禁欲，不近女色
- **适合题材**：仙侠/古言
- **人设要点**：
  - 万年冰山只为一人融化
  - 禁欲是表象，深情是内核
  - 动情后的反差是最大看点
- **经典台词风格**："……""不必""你……与其他人不同"
- **写作禁忌**：不能冷到读者没耐心；动情要有铺垫不能突然

## CP模式

### 甜宠

- **模式特征**：全程高甜，冲突来自外部
- **感情线**：从甜到更甜，螺旋上升
- **冲突来源**：外部势力（情敌/家族/误会），不伤CP感情
- **节奏**：日常甜度60%+，小波折20%，解决+更甜20%
- **关键**：甜不能腻，要有变化；每章至少一个心动瞬间

### 虐恋

- **模式特征**：甜蜜→误会→虐心→和好
- **感情线**：先甜后虐再甜，U型曲线
- **冲突来源**：CP之间的误解/背叛/立场对立
- **节奏**：甜30%→虐40%→和好30%
- **关键**：虐的前提是甜够；和好要有过程不能一秒原谅

### 双向奔赴

- **模式特征**：双方都有成长，互相成就
- **感情线**：平行成长→交汇→携手
- **冲突来源**：各自成长路上的障碍
- **节奏**：各自线40%+交汇线20%+携手线40%
- **关键**：双方戏份要均衡；成长要同步不能一个飞一个停

### 追妻火葬场

- **模式特征**：男追女，先虐后甜
- **感情线**：男伤女→女离开→男追回
- **冲突来源**：男主的前期伤害+女主的信任危机
- **节奏**：伤害20%→离开10%→追回50%→和好20%
- **关键**：追的过程要够苦够诚；女主不能太容易原谅

### 先婚后爱

- **模式特征**：从陌生到相爱
- **感情线**：陌生→磨合→心动→相爱
- **冲突来源**：性格差异/生活习惯/外部阻力
- **节奏**：陌生30%→磨合30%→心动20%→相爱20%
- **关键**：婚后日常要写好；心动瞬间要有铺垫

## 配角设计

### 绿茶

- **功能**：制造冲突、衬托女主、推动剧情
- **设计要点**：表面温柔善良实则心机深，善于在男人面前装可怜
- **下场**：必须被揭穿/打脸，读者最期待的场景
- **注意**：不能太蠢，否则打脸没快感；要有自己的逻辑和动机

### 白月光

- **功能**：制造误会、考验CP、推动感情
- **设计要点**：男主心中的完美前任，是女主最大的不安来源
- **走向**：要么真完美让女主成长，要么假完美被揭穿
- **注意**：不能喧宾夺主；存在感要恰到好处

### 闺蜜

- **功能**：情感支撑、剧情助攻、调节氛围
- **设计要点**：真心为女主好，有自己的感情线
- **注意**：不能抢女主风头；不能变成工具人

### 婆婆

- **功能**：制造阻力、考验CP、反映社会现实
- **设计要点**：有自己的一套逻辑，不是纯粹的恶
- **走向**：要么被女主感化，要么被男主挡回去
- **注意**：不能脸谱化；要有让人理解的立场

### 情敌

- **功能**：催化感情、制造危机、衬托女主
- **设计要点**：有自己的魅力，不是纯粹的炮灰
- **注意**：高级情敌让读者也纠结，低级情敌只是打脸素材

## 女频角色关系网设计法

### 设计原则

1. **以女主为中心** — 所有人物关系围绕女主展开
2. **每条关系线都有功能** — 推动剧情/衬托人设/制造冲突
3. **关系网要有层次** — 核心圈（CP/闺蜜）→ 中间圈（家族/同事）→ 外围圈（路人/过场）
4. **关系要动态变化** — 敌人变盟友、朋友变情敌，保持新鲜感

### 设计步骤

**第一步：确定核心关系**
- 女主与男主的关系模式（见CP模式）
- 这是全书的主轴，其他关系都围绕它

**第二步：设计对立关系**
- 谁是女主的主要对手？（绿茶/婆婆/情敌）
- 对手的动机要合理，不能为反对而反对

**第三步：设计支撑关系**
- 谁是女主的盟友？（闺蜜/家人/贵人）
- 盟友要有自己的故事，不能只是工具人

**第四步：设计催化关系**
- 谁能推动CP感情发展？（白月光/情敌/助攻）
- 催化剂要适时出现，不能太早也不能太晚

**第五步：画出关系网**
- 用图表标明每对关系：友好/对立/暧昧/利用
- 检查是否有孤立角色（与主线无关的人要删掉）
- 检查关系密度（核心圈不超过5人）

### 关系网常见模式

```
        [白月光]——暧昧——[男主]——CP——[女主]——闺蜜——[闺蜜]
           |                |                |
         制造不安         追妻/宠妻        支撑/助攻
           |                |                |
        [情敌]          [婆婆/家族]      [绿茶]
```

### 关系网动态设计

- 每20-30章引入一个新关系或改变一个旧关系
- 关系变化要有触发事件，不能无缘无故
- 好的关系可以变坏，坏的关系可以变好
- 每次关系变化都要带来新的剧情可能

# 第三部分：开头设计

## 女频黄金一章法则

### 与男频的核心差异

男频黄金一章强调**冲突代入**——用最激烈的冲突抓住读者。
女频黄金一章强调**情感代入**——用最强烈的情感共鸣抓住读者。

| 维度 | 男频 | 女频 |
|------|------|------|
| 第一优先 | 冲突/悬念 | 情感共鸣 |
| 代入方式 | 我要变强 | 我要被爱/我要改命 |
| 开篇核心 | 金手指+危机 | 人设+情感困境 |
| 读者期待 | 这主角怎么翻盘 | 这女主怎么破局 |

### 女频黄金一章要素

1. **情感钩子** — 开篇即建立情感共鸣，让读者"心疼"或"好奇"
2. **人设亮相** — 女主核心特质在前500字内展现
3. **困境呈现** — 女主面临的情感/生存困境
4. **转折暗示** — 暗示"这次不一样"，给读者希望
5. **CP初遇或暗示** — 即使不立刻出场，也要有男主的影子

### 女频第一章字数分配参考

| 内容 | 占比 | 说明 |
|------|------|------|
| 情感钩子/困境 | 30% | 开篇即抓住情感 |
| 人设展示 | 25% | 通过行为展现性格 |
| 转折/金手指 | 20% | 给读者希望 |
| CP暗示/初遇 | 15% | 感情线启动 |
| 钩子/悬念 | 10% | 让读者想看下一章 |

## 女频开局模板

### 穿书开局

**核心逻辑**：发现自己穿进小说→知道剧情→决定改命

**开篇节奏**：
1. 穿越瞬间（300字内完成穿越）
2. 发现自己是谁（原书角色/炮灰/反派）
3. 知道原书剧情（关键信息：谁会害我/我的结局）
4. 做出决定（这次我要改命）

**示例结构**：
```
醒来→发现环境不对→记忆涌入→我穿书了→原主结局惨→不行，这次我要改
```

**关键技法**：
- 穿越过程要快，不要纠结"我怎么来的"
- 原书剧情只交代关键信息，不要大段复述
- 女主的决定要果断，不能犹豫太久
- 立刻给出第一个改命行动的暗示

**常见雷区**：穿越过程写太多、原书剧情大段复述、女主犹豫不决

### 重生开局

**核心逻辑**：前世惨死→重生到关键节点→这次不一样

**开篇节奏**：
1. 前世死亡瞬间（画面感强，情感冲击）
2. 重生醒来（时间跳跃要清晰）
3. 回到关键节点（这是改变命运的起点）
4. 前世记忆带来的信息差（我知道接下来会发生什么）

**示例结构**：
```
前世惨死画面→重生醒来→确认时间→前世记忆→这次绝不重蹈覆辙
```

**关键技法**：
- 前世死亡要惨但篇幅要短（500字以内）
- 重生后的第一个行动要体现"这次不一样"
- 前世记忆是金手指，但不能全知全能
- 立刻给出第一个改命行动

**常见雷区**：前世写太多、重生后还在回忆、改命行动太慢

### 替嫁开局

**核心逻辑**：被迫替嫁→对方竟然是→意想不到的发展

**开篇节奏**：
1. 替嫁处境（被逼无奈/为家人牺牲）
2. 嫁入对方家（忐忑/不安）
3. 发现对方不是传闻那样（反转）
4. 意想不到的发展（对方竟然……）

**示例结构**：
```
被迫替嫁→传闻中的丈夫很可怕→嫁过去→发现他竟然→？
```

**关键技法**：
- 替嫁的原因要让读者心疼
- 传闻与现实的反差是核心看点
- 男主的真实面目要逐步揭开
- 第一个反转要在第一章内出现

**常见雷区**：替嫁原因不合理、男主反转太快太彻底

### 退婚开局

**核心逻辑**：被退婚→转身遇到更好的→前任后悔

**开篇节奏**：
1. 退婚现场（屈辱/不甘）
2. 女主反应（不哭不闹/反而洒脱）
3. 转身遇到更好的（命运转折）
4. 前任开始后悔（伏笔）

**示例结构**：
```
当众被退婚→所有人看笑话→女主不卑不亢→转身遇到真命→前任：？
```

**关键技法**：
- 退婚现场要写得让读者替女主不平
- 女主的反应要超出预期（不哭反笑/洒脱离开）
- 真命出场要有仪式感
- 前任的后悔要延后，不能太快

**常见雷区**：女主退婚后哭哭啼啼、真命出场太突兀

### 和亲开局

**核心逻辑**：被迫和亲→对方竟然→从敌到爱

**开篇节奏**：
1. 和亲使命（国家需要/家族安排）
2. 前往异国（忐忑/不安/决心）
3. 初见和亲对象（敌意/试探）
4. 发现对方不是想象中那样

**示例结构**：
```
被迫和亲→传闻中的野蛮王→初见→他竟然→从敌意到好奇
```

**关键技法**：
- 和亲的牺牲感要写出来
- 文化差异是天然冲突源
- 从敌到爱的过程要自然
- 政治博弈增加深度

**常见雷区**：男主太快变温柔、政治线太复杂喧宾夺主

### 系统开局

**核心逻辑**：获得系统→任务→奖励→改变命运

**开篇节奏**：
1. 系统降临（意外/绑定）
2. 系统功能介绍（简洁明了）
3. 第一个任务（立刻开始）
4. 完成任务/获得奖励（爽感启动）

**示例结构**：
```
困境中→系统绑定→了解功能→第一个任务→完成→奖励→命运开始改变
```

**关键技法**：
- 系统功能要简洁，不要大段说明
- 第一个任务要简单但有意义
- 奖励要立刻体现效果
- 系统是工具不是主角，女主的主动性要强

**常见雷区**：系统说明太长、女主被系统牵着走、任务太机械

## 女频开头信息释放优先级

女频开头的核心原则：**情感共鸣 > 人设 > 金手指 > 世界观**

### 优先级排序

| 优先级 | 信息类型 | 释放时机 | 说明 |
|--------|---------|---------|------|
| 1 | 情感共鸣 | 第1段 | 让读者立刻产生情感连接 |
| 2 | 人设 | 前500字 | 女主核心特质必须尽早展现 |
| 3 | 困境 | 前1000字 | 女主面临的问题 |
| 4 | 金手指 | 前2000字 | 改变命运的契机 |
| 5 | 世界观 | 穿插释放 | 不要大段介绍，融入剧情 |
| 6 | 配角 | 按需出场 | 谁先出场取决于谁对开篇最有用 |

### 信息释放原则

- **情感先行**：先让读者"心疼"或"好奇"，再交代其他
- **人设即剧情**：通过女主的行为展现人设，不要旁白介绍
- **金手指要快**：第一章内必须出现，但不需要完全解释
- **世界观后置**：读者不需要了解整个世界，只需要了解女主的世界
- **配角按需**：谁对当前剧情有用谁先出场

## 女频开头禁忌

### 三大禁忌

1. **不要让女主太弱**
   - 可以身处困境，但不能没有主见
   - 可以暂时无力，但不能放弃
   - 读者要看到希望的种子

2. **不要让女主太蠢**
   - 信息差可以，降智不行
   - 不知道可以，知道还往坑里跳不行
   - 善良可以，圣母不行

3. **不要让女主太作**
   - 有脾气可以，无理取闹不行
   - 耍小性子可以，作到让人烦不行
   - 读者要心疼不要心烦

### 其他常见问题

- **大段背景介绍** — 女频读者比男频更没耐心看设定
- **前世/原书剧情写太多** — 点到为止，后续穿插
- **男主迟迟不出场** — 第一章至少要有男主的影子
- **开篇太平** — 没有情感冲击，读者不会留到第二章
- **人设与行为矛盾** — 说了是女强人结果开篇就哭
- **金手指太晚出现** — 第一章必须给读者希望

## 女频钩子设计

### 核心原则：情感钩子 > 剧情钩子

男频用悬念留人（"他能不能赢？"），女频用情感留人（"她会不会被爱？"）。

### 情感钩子类型

| 钩子类型 | 示例 | 效果 |
|---------|------|------|
| 心疼钩 | 女主独自承受一切 | 读者心疼想看有人心疼她 |
| 期待钩 | 男主出现/暗示 | 读者期待CP互动 |
| 反差钩 | 传闻vs现实 | 读者好奇真相 |
| 不甘钩 | 被欺负/被误解 | 读者想看打脸/翻盘 |
| 选择钩 | 两难抉择 | 读者想知道她怎么选 |

### 章末钩子设计

每章结尾必须有一个钩子，让读者忍不住点下一章：

- **甜宠文**：CP互动的甜蜜悬念（"他竟然说……"）
- **虐恋文**：即将到来的虐点暗示（"她不知道的是……"）
- **宫斗/宅斗**：下一个阴谋的苗头（"有人开始动了"）
- **重生文**：前世记忆的预警（"这个时间，前世发生了……"）
- **快穿/穿书**：新世界的悬念（"系统提示：下一个世界即将开启"）

### 钩子密度要求

- 第一章：至少3个钩子（情感+人设+悬念）
- 前5章：每章至少2个钩子
- 日常章节：每章至少1个钩子
- 高潮章节：钩子可以更密，但不要连续虐不给人喘息
