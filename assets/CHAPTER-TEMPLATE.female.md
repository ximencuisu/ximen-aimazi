# 女频小说章节模板

适用于女频小说（言情、甜宠、穿越、宫斗等）的章节生成模板。

---

## 标准章节模板

```markdown
# 第XX章 [章节名]

> **本章概要**：一句话概括本章核心事件
> **情感走向**：甜蜜升温 / 虐心 / 误会 / 和好 / 突破
> **CP进度**：[初遇/相识/暧昧/表白/确认/升温/危机/和好]
> **情绪曲线**：[甜到齁 / 虐到哭 / 笑中带泪 / 欲罢不能]

---

（正文内容，2500-3000字）

（开篇：承接上章 / 场景切入 / 偶遇/对峙）

（发展：互动增加 / 误会产生 / 暗生情愫 / 危机出现）

（高潮：真情流露 / 误会解开 / 甜蜜互动 / 情感爆发）

（收束：结果呈现 / 关系变化 / 留下悬念）

---

> **章末钩子**：[暧昧互动/未解之谜/下章预告]
> **甜度指数**：⭐⭐⭐⭐⭐（5星满）
```

---

## 女频章节质量检查

### 连贯性检查
- [ ] 与上一章情感走向衔接自然
- [ ] 角色状态与 `CHARACTERS.md` 一致
- [ ] 男主行为是否符合其人设（霸道/温柔/腹黑/阳光）
- [ ] 女主反应是否符合其性格（坚强/软萌/聪慧/独立）

### 情感检查
- [ ] 有明确的情感进展或转折
- [ ] 男女主互动有张力（眼神/动作/对话）
- [ ] 甜点场景是否够甜（心跳/脸红/肢体接触）
- [ ] 虐点场景是否够虐（误会/分离/受伤/心酸）

### 读者体验检查
- [ ] 代入感强，读者能感受女主的心情
- [ ] 男主人设有魅力（苏点）
- [ ] 配角有特色（闺蜜/情敌/助攻）
- [ ] 章末钩子让人想看下一章

## 女频爽点设计

### 甜宠类爽点

| 爽点类型 | 描述 | 示例 |
|---------|------|------|
| 霸道总裁 | 男主强势宠溺 | "我的女人，谁敢动一下试试" |
| 暗戳戳甜 | 不明显但很甜 | 偷偷关注/默默付出/故意接近 |
| 公开撒糖 | 当众宣示主权 | 当众牵手/介绍身份/霸气护短 |
| 身体接触 | 心跳加速 | 壁咚/公主抱/十指紧扣/额头吻 |

### 虐心情节

| 虐点类型 | 描述 | 示例 |
|---------|------|------|
| 误会 | 信息不对称 | 看到不该看的/听到不该听的 |
| 分离 | 客观原因分开 | 家族反对/身份差距/被迫嫁人 |
| 受伤 | 身体或心灵 | 为对方受伤/独自承受痛苦 |
| 心酸 | 暗恋/单恋 | 默默付出/得不到回应 |

### 逆袭爽点

| 爽点类型 | 描述 | 示例 |
|---------|------|------|
| 打脸 | 让看不起的人后悔 | 身份揭秘/能力展示/价值飙升 |
| 翻身 | 从被欺负到反击 | 揭露真相/夺回主权 |
| 选择 | 让对方后悔 | 选别人/变得更优秀 |

## 女频节奏设计

### 感情线节奏

```
每 2-3 章：小互动（眼神/小动作）
每 4-6 章：暧昧升温（单独相处/暗生情愫）
每 8-10 章：大进展（表白/确认关系/亲密接触）
每 15-20 章：感情危机（误会/分离/情敌）
每 25-30 章：感情升华（化解危机/修成正果）
```

### 宫斗/职场节奏

```
每 1-2 章：小冲突（有人找茬）
每 3-5 章：化解危机 + 展示能力
每 8-10 章：大危机（陷害/打压）
每 15-20 章：反击 + 打脸对手
每 25-30 章：终极对决 + 登上巅峰
```
