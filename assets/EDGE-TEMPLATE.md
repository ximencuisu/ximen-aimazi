### 【风格A】含蓄暗示型

```markdown
## 核心特点
- 点到为止，留白给读者想象
- 暗示为主，不直接描写
- 氛围感强，暧昧朦胧
- 适合：言情、都市、仙侠

## 描写要求
- 用环境烘托氛围（烛光、月色、雨声）
- 用微动作暗示（指尖触碰、呼吸变化、眼神交汇）
- 用心理活动制造张力（心跳加速、脸红、回避目光）
- 关键时刻戛然而止，留悬念

## 允许内容
- 拥抱、接吻（简短描写）
- 暧昧对话（双关语、暗示）
- 身体接触（牵手、搂腰）
- 氛围营造（独处、暧昧环境）

## 禁止内容
- 直接描写性行为
- 详细描写身体部位
- 露骨的对话
- 超过3行的亲密描写

## 示例句式
- "她的气息近在咫尺，他喉结滚动了一下。"
- "月光落在她肩上，他移开了视线。"
- "门关上了。房间里只剩下两个人的呼吸声。"
```

### 【风格B】直白热烈型

```markdown
## 核心特点
- 直接描写，不绕弯子
- 情欲张力强烈
- 身体感受明确
- 适合：都市、重生、系统流

## 描写要求
- 直接描写身体反应（心跳、体温、触感）
- 对话可以大胆（调情、挑逗）
- 动作描写具体但不露骨
- 情绪张力要足

## 允许内容
- 激烈的拥抱和接吻
- 大胆的暧昧对话
- 身体接触（抚摸、贴近）
- 暗示性行为发生（但不描写过程）

## 禁止内容
- 详细的性行为过程描写
- 过于露骨的身体部位描写
- 超过平台尺度的内容
- 强迫/非自愿情节

## 示例句式
- "他把她按在墙上，吻了下去。"
- "她的手指划过他的胸膛，他抓住了她的手腕。"
- "衣服落在地上，灯熄了。"
```

### 【风格C】文艺唯美型

```markdown
## 核心特点
- 用诗意语言描写情欲
- 美化肉体，升华情感
- 意象丰富，意境深远
- 适合：仙侠、古风、言情

## 描写要求
- 用自然意象比喻（花、水、月、云）
- 情感与肉体交融
- 语言优美，有文学性
- 节奏舒缓，有仪式感

## 允许内容
- 唯美的亲密描写
- 诗意的暧昧对话
- 朦胧的身体描写
- 情感升华的情节

## 禁止内容
- 粗俗的语言
- 过于直白的描写
- 破坏美感的细节
- 纯粹的生理描写

## 示例句式
- "如春水化冰，肌肤相触的瞬间，万物静默。"
- "她的长发散落在他掌心，像月光流泻。"
- "两道身影在纱幔后重叠，烛火摇曳。"
```

### 【风格D】暴力美学型

```markdown
## 核心特点
- 情欲与危险交织
- 张力来自对抗和征服
- 野性、原始、激烈
- 适合：玄幻、修仙、黑道

## 描写要求
- 强调力量感和控制欲
- 对抗中的暧昧
- 危险带来的情欲张力
- 角色间的权力博弈

## 允许内容
- 激烈的肢体接触
- 带有征服意味的互动
- 危险环境中的暧昧
- 强势的对话和动作

## 禁止内容
- 真正的暴力伤害
- 非自愿情节
- 超过尺度的描写
- 虐待倾向

## 示例句式
- "他掐住她的下巴，力道不重，却不容抗拒。"
- "血从他嘴角流下，她舔掉了那滴血。"
- "他们像两头困兽，在生死边缘试探彼此。"
```

### 【风格E】纯情青涩型

```markdown
## 核心特点
- 青春感、纯真感
- 小心翼翼的靠近
- 羞涩和甜蜜
- 适合：校园、青春、初恋

## 描写要求
- 强调心理活动（紧张、害羞、期待）
- 身体接触要克制（碰一下手都会脸红）
- 对话要青涩（结巴、词不达意）
- 甜蜜感大于情欲感

## 允许内容
- 牵手、拥抱（简短）
- 偷看、暗恋心理
- 青涩的表白
- 纯真的亲吻

## 禁止内容
- 任何暗示性行为的内容
- 过于成熟的情节
- 露骨的身体描写
- 复杂的情感纠葛

## 示例句式
- "他的手碰到她的，像触电一样缩了回来。"
- "她低着头，耳朵红得像熟透的苹果。"
- "他们在走廊上擦肩而过，心跳快得像要跳出胸口。"
```

## 使用说明

1. 选择或自定义一种风格
2. 取消注释对应的风格配置（删除 ```markdown 和 ``` 之间的注释标记）
3. 保存此文件
4. 在生成擦边情节正文时，系统会自动载入此风格配置
5. 所有擦边描写都必须严格按照此风格要求执行

## 擦边情节分级标准

| 等级 | 描写范围 | 字数占比 | 平台安全度 |
|------|---------|---------|-----------|
| 含蓄型 | 暧昧暗示，无直接描写 | 1-2% | 高 |
| 直白型 | 暗示性行为发生，不描写过程 | 3-5% | 中 |
| 文艺型 | 唯美描写，有朦胧感 | 2-4% | 中高 |
| 暴力型 | 张力描写，对抗中的暧昧 | 3-5% | 中 |
| 纯情型 | 纯真接触，无情欲色彩 | 1-2% | 高 |

## 章节擦边分布建议

根据"擦边等级"参数自动安排：

### 少量（5-10%）

```
- 每10-20章安排1次暧昧互动
- 以氛围暗示为主
- 关键词：眼神、呼吸、触碰、独处
```

### 中量（15-25%）

```
- 每4-6章安排1次擦边情节
- 有实质互动但不越界
- 关键词：拥抱、亲吻、暧昧对话、身体接触
```

### 大量（30-40%）

```
- 每2-3章安排1次擦边情节
- 频繁出现，尺度较大
- 关键词：情欲张力、亲密行为、暗示性行为
```
