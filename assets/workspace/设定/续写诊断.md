# 续写诊断

> 续写专用：诊断卡文/断更点的类型与根因。CP2 阶段建立，CP3、Phase 7 续写时读取。
> 卡点类型分 5 类，先定位类型再制定方案。

## 卡点诊断

| 卡点类型 | 是否命中 | 根因分析 | 涉及章节/角色/线 |
|----------|----------|----------|------------------|
| 结构 | ☐ | 待填 | |
| 人物 | ☐ | 待填 | |
| 资源 | ☐ | 待填 | |
| 信息 | ☐ | 待填 | |
| 情绪 | ☐ | 待填 | |

> 类型说明：
> - 结构：情节走不下去/伏笔无法回收/节奏失衡
> - 人物：角色动机断裂/人设崩坏/关系无法推进
> - 资源：战力/物品/金钱体系失衡或枯竭
> - 信息：信息差失控/读者已知/角色未知错位
> - 情绪：情绪曲线断裂/爽点兑现不足/虐点过载
