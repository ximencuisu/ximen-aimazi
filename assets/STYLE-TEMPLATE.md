# 文风提示词配置

本文档用于定义小说写作的文风要求。在生成正文前会自动载入此配置。

---

## 当前激活文风

**ACTIVE_STYLE=无**

> AI 读写说明：选择文风后，将上方 `无` 替换为对应选项（A/B/C/D/E/自定义/混搭）。选择\u201c上次\u201d时直接读取此行的值。

---

### 【文风A】热血爽文风

```markdown
## 核心特点
- 节奏快，爽点密集
- 打脸干脆利落，不留余地
- 升级突破描写要燃
- 主角气场强大，碾压对手

## 句式要求
- 短句为主，干净利落
- 动作描写要快：「他出手了。」而不是「他缓缓抬起手，准备出手。」
- 对话简洁有力，不废话
- 避免长段心理描写

## 禁止事项
- 禁止拖沓的环境描写（超过2句）
- 禁止过多的旁白解释
- 禁止主角犹豫不决
- 禁止反派废话连篇

## 爽点节奏
- 每500字至少一个爽点
- 打脸要彻底，不要给反派喘息机会
- 突破要震撼，要有围观群众惊叹
```

---

### 【文风B】都市现实风

```markdown
## 核心特点
- 贴近现实，有烟火气
- 对话自然，像真人说话
- 细节真实，有生活感
- 人物有缺点，不完美

## 句式要求
- 口语化表达
- 适当使用方言或俚语
- 心理活动要真实，不矫情
- 场景描写要有画面感

## 禁止事项
- 禁止过于中二的台词
- 禁止不切实际的金手指设定
- 禁止脸谱化的反派
- 禁止过度装逼

## 情感节奏
- 感情线要循序渐进
- 冲突要有合理动机
- 转折要有铺垫
```

---

### 【文风C】古风仙侠风

```markdown
## 核心特点
- 文字有古韵，但不晦涩
- 仙气飘飘，意境深远
- 战斗场面要写意
- 人物气质出尘

## 句式要求
- 四字词语适当使用
- 可以用对偶、排比
- 描写要有画面感和意境
- 对话可以文雅一些

## 禁止事项
- 禁止过于现代化的表达
- 禁止网络流行语
- 禁止过于血腥暴力的描写
- 禁止完全文言文化（影响阅读）

## 特色元素
- 法宝、丹药、阵法的描写要古雅
- 境界突破要有天象异动
- 人物名字要有仙气
```

---

### 【文风D】悬疑惊悚风

```markdown
## 核心特点
- 氛围压抑，悬念迭起
- 节奏张弛有度
- 细节暗示，伏笔密布
- 反转出人意料

## 句式要求
- 多用短句制造紧张感
- 环境描写要阴暗压抑
- 心理描写要细思极恐
- 对话要有潜台词

## 禁止事项
- 禁止过早揭示真相
- 禁止无意义的惊吓（jump scare）
- 禁止逻辑漏洞
- 禁止主角过于被动

## 悬念节奏
- 每章结尾必须有悬念
- 线索要分散埋设
- 真相要层层揭开
```

---

### 【文风E】轻松日常风

```markdown
## 核心特点
- 轻松幽默，读起来愉快
- 日常琐碎但有趣
- 角色互动有爱
- 没有太大压力

## 句式要求
- 口语化，像朋友聊天
- 可以用吐槽和玩梗
- 心理活动可以搞笑
- 对话要活泼

## 禁止事项
- 禁止过于沉重的剧情
- 禁止虐心情节
- 禁止复杂的政治斗争
- 禁止黑暗的结局

## 节奏把控
- 剧情推进可以慢一些
- 重点放在人物互动
- 日常小事要有趣
```

## 自定义文风

如果以上预设都不符合需求，在下方自定义你的文风：

```markdown
## 核心特点
- （填写你的文风特点）

## 句式要求
- （填写句式要求）

## 禁止事项
- （填写禁止事项）

## 其他要求
- （填写其他特殊要求）
```

---

## 文风混搭（高级）

可以组合多种文风元素：

```markdown
## 主文风
热血爽文风

## 混搭元素
- 都市现实风的对话自然感
- 古风仙侠风的意境描写

## 混搭比例
- 80% 主文风
- 20% 混搭元素
```

---

## 使用说明

1. 选择或自定义一种文风
2. 取消注释对应的文风配置（删除 ```markdown 和 ``` 之间的注释标记）
3. 保存此文件
4. 在生成正文时，系统会自动载入此文风配置
5. 所有正文都会按照此文风要求来写

---

## 文风切换

如果需要在不同章节使用不同文风：

```markdown
## 章节文风覆盖

| 章节范围 | 文风 |
|---------|------|
| 第1-10章 | 热血爽文风 |
| 第11-20章 | 都市现实风 |
| 第21章+ | 自定义文风X |
```
